# 掼蛋快速入门

## 概述

作为新手，掌握掼蛋的基本规则是第一步，接下来需逐步提升实战能力。

## 快速入门步骤

### 第一步：熟悉牌型

熟练掌握各类牌型及其组合方式，确保快速反应。

**核心牌型**（平台标准变量，依据江苏掼蛋规则）：
- **单张 (Single)**：任意一张牌
- **对子 (Pair)**：两张牌点相同的牌，包括两张大王(`R`)和两张小王(`B`)
- **三连对 (ThreePair)**：三个牌点相邻的对子
- **三同张 (Trips)**：三张牌点相同的牌
- **二连三 (TwoTrips)**：两个牌点相邻的三同张牌（俗称"钢板"）
- **三带对 (ThreeWithTwo)**：一个三同张和一个对子
- **顺子 (Straight)**：五张牌点相邻的单张
- **同花顺 (StraightFlush)**：花色相同的顺子
- **炸弹 (Bomb)**：四张(含)以上牌点相同的单张
- **王炸 (Bomb)**：大王(`R`)、小王(`B`)各两张

### 第二步：理解比赛流程

- **局**：一方从2级开始打到A级胜出，称为一局
- **场**：每次完整比赛称为一场
- **轮**：所有参赛牌手普遍碰面，称为一轮

### 第三步：掌握基本策略

- **首圈领牌权**：首圈领谁先出牌对于整局比赛的节奏有显著影响，需合理利用贡牌与还牌策略
- **出牌策略**：合理安排出牌顺序，尽量压制对手，提高自己和搭档的胜率
- **灵活应对**：根据实际牌局变化调整策略，灵活变通，提升团队合作
- **升级与打级**：上游和二游要及时沟通，确保尽早升级，掌握比赛主动权

### 第四步：练习技巧

- **练习技巧**：通过与不同水平的玩家对局，积累丰富的实战经验
- **观看教学视频**：学习高手的出牌策略和战术，取长补短
- **多加练习**：掼蛋虽简单易学，但要想精通，需反复练习

## 新手常见问题

1. **如何判断牌力？** 需要学习牌力评估方法
2. **何时主攻何时助攻？** 需要理解主攻与助攻的职责
3. **如何配合对家？** 需要掌握团队配合技巧
