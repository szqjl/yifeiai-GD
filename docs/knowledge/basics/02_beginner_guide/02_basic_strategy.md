# 掼蛋基础策略

## 概述

比赛中的掼蛋具有层级性和策略性，选手不仅要熟悉规则，还需灵活运用技巧。以下是一些关键的比赛规则和策略建议。

## 核心策略要点

### 1. 首圈领牌权

**重要性**：首圈领谁先出牌对于整局比赛的节奏有显著影响

**策略要点**：
- 需合理利用贡牌与还牌策略
- 根据手牌情况决定是否争取首圈领牌权
- 主攻方应争取首圈领牌权，助攻方可以配合

### 2. 出牌策略

**核心原则**：合理安排出牌顺序，尽量压制对手，提高自己和搭档的胜率

**策略要点**：
- 根据牌力判断主攻或助攻角色
- 主攻方：优先处理弱路，快速出牌
- 助攻方：配合主攻，控制对手，送对家
- 灵活调整出牌顺序，避免被对手控制

### 3. 灵活应对

**核心原则**：根据实际牌局变化调整策略，灵活变通，提升团队合作

**策略要点**：
- 观察对手出牌习惯和牌力
- 根据牌局变化及时调整策略
- 与对家保持默契，相互配合
- 不要死板执行既定策略

### 4. 升级与打级

**核心原则**：上游和二游要及时沟通，确保尽早升级，掌握比赛主动权

**升级规则**（依据江苏掼蛋规则）：
- 只有上游方可以升级
- 对家获得二游升3级
- 对家获得三游升2级
- 对家获得下游升1级
- A级必打，获得上游，且对家获得二游或三游，才能打过A级

**策略要点**：
- 级数从2至A共有13个级数，逐级晋升
- 及时升级可以掌握比赛主动权
- 需要与对家配合，确保升级成功
- A级是必打级别，需要特别注意

## 应用场景

### 场景1：开局阶段

- **主攻方**：争取首圈领牌权，优先处理弱路
- **助攻方**：配合主攻，观察对手，准备控制

### 场景2：中局阶段

- **主攻方**：继续冲刺，控制出牌节奏
- **助攻方**：送对家，卡对手，配合主攻

### 场景3：残局阶段

- **主攻方**：快速出牌，争取头游
- **助攻方**：全力配合，送对家，阻止对手

## 注意事项

- 不要盲目出牌，要根据牌局情况灵活调整
- 要与对家保持沟通和配合
- 不要死板执行策略，要根据实际情况变化
- 要观察对手的出牌习惯和牌力
