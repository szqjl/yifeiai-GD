# 牌张分配规则

## 概述
掼蛋游戏的牌张分配规则，包括牌数、发牌方式等基础规则。

## 牌张数量
- **全副牌**：108张（两副扑克牌）
- **每位牌手**：27张牌
- **牌型组成**：54张 × 2 = 108张

## 发牌规则
- **发牌顺序**：按座位顺序依次抓牌或发牌，通常是逆时针方向。
- **发牌方式**：每人27张，依次发完或抓完
- **底牌**：无底牌（所有牌都发给玩家）

## 牌张组成
- **普通牌**：A、2-9、T(10)、J、Q、K（每种4张，共52张 × 2 = 104张）
- **王牌**：大王(R)、小王(B)（各2张，共4张）
- **总计**：104 + 4 = 108张

## 注意事项
- 牌张分配必须确保每位玩家获得27张牌
- 发牌过程必须随机，确保公平性
