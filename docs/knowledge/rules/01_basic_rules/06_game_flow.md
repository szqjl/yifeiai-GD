# 游戏流程规则

## 概述
掼蛋游戏的完整流程规则，包括游戏阶段、出牌顺序等。

## 游戏阶段（stage）

### 1. 开始阶段 (beginning)
- **说明**：游戏开始，发牌完成
- **状态**：等待首攻玩家出牌
- **平台变量**: `stage: "beginning"`

### 2. 游戏阶段 (play)
- **说明**：正常游戏进行中
- **状态**：玩家依次出牌或PASS
- **平台变量**: `stage: "play"`

### 3. 进贡阶段 (tribute)
- **说明**：上一局结束后，下游向上游进贡
- **状态**：等待进贡玩家出牌
- **平台变量**: `stage: "tribute"`

### 4. 还贡阶段 (back)
- **说明**：上游向下游还贡
- **状态**：等待还贡玩家出牌
- **平台变量**: `stage: "back"`

### 5. 局结束 (episodeOver)
- **说明**：一局游戏结束
- **状态**：等待下一局开始
- **平台变量**: `stage: "episodeOver"`

### 6. 游戏结束 (gameOver)
- **说明**：整场游戏结束
- **状态**：游戏完全结束
- **平台变量**: `stage: "gameOver"`

## 出牌流程

### 首圈领出规则（依据江苏掼蛋规则）

1. **首副牌**：
   - 由南家切牌并翻出一张牌（如翻出的牌为王牌或红心2，则重切重翻）
   - 从南家开始，依翻出牌张的牌点，按逆时针方向点数出首抓者
   - **抓到翻出牌张者首圈领出牌**

2. **从第二副牌开始**（依据江苏掼蛋规则225-235行）：
   - **正常进贡**：首圈由下游领出牌
   - **双下进贡**：还牌后，由进贡给上游者首圈领出牌
   - **抗贡**（抓到两个大王）：首圈由上游领出牌

### 出牌顺序

1. **出牌顺序**：按座位顺序逆时针依次出牌（0→1→2→3→0...）
2. **压牌规则**：必须出能压制当前牌型的牌，或选择PASS
3. **接风**：如果连续3个玩家PASS，最后一个出牌的玩家获得接风权（对家接风）

## 游戏术语
- **一副（局）牌**：从抓牌开始到一方打完结束
- **一手牌**：牌手一次打出的一组牌
- **一圈牌**：四位牌手依次出牌直到无人出牌
- **上游/二游/三游/下游**：出完牌的顺序
