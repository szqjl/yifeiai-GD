# 进贡规则

## 概述
掼蛋游戏中的进贡规则，包括进贡条件、进贡方式、还贡规则等。本文档整合了江苏掼蛋规则和平台使用说明书的详细内容。

## 贡牌规则

### 基本进贡规则
- **适用时机**：从第二副牌开始，在出牌前由下游向上游或双下方向对方进贡牌张
- **进贡方式**：
  - 由上副牌的下游向上游进贡全手牌中牌点最大的一张牌（红心级牌除外）
  - 上游选择全手牌中任意一张牌点10(含)以下的牌还给下游。如上游手中牌点均超过10，则选择牌点最小的牌还牌
  - 首圈由下游领出牌
- **平台变量**：`tribute`（进贡）、`back`（还贡）
- **平台阶段**：`stage: "tribute"`（进贡阶段）

### 双下进贡（依据江苏掼蛋规则230-233行）
- **适用条件**：如上副牌出现双下，则双下方两位牌手均向对方进贡
- **进贡对象**：三游向上游进贡，下游向二游进贡
- **还牌规则**：
  - 上游选择牌点较大的牌还给三游
  - 对家（二游）选择牌点较小的牌还给下游
- **还牌方式**：还牌时，应将牌面向下，两位进贡者同时亮牌。如贡牌牌点相同，则按顺时针方向进贡，对应还牌
- **首圈领牌权**：还牌后，由进贡给上游者首圈领出牌（即三游首圈领出牌）

### 抗贡规则
- **抗贡条件**：如应进贡者（下游或双下方）抓到两个大王(`R`)，则不须进贡（抗贡）
- **抗贡结果**：首圈由上游领出牌
- **平台处理**：系统会自动识别抗贡情况

### 时间限制
- **进贡时间**：进贡者应在抓完牌后20秒(含)以内贡牌
- **还牌时间**：还牌者须在贡牌后40秒(含)以内还牌

## 还贡规则

### 基本还贡规则
- **还贡对象**：上游向进贡者还贡
- **还贡牌选择**：
  - 正常进贡：上游选择全手牌中任意一张牌点10(含)以下的牌还给下游。如上游手中牌点均超过10，则选择牌点最小的牌还牌
  - 双下进贡：上游选择牌点较大的牌还给三游，对家（二游）选择牌点较小的牌还给下游
- **还牌方式**：还牌时，应将牌面向下，两位进贡者同时亮牌
- **平台变量**：`back`（还贡）
- **平台阶段**：`stage: "back"`（还贡阶段）

## 注意事项
- 进贡规则必须严格按照平台标准执行
- v1006版本调整了抗贡规则，与比赛版规则一致
- 红心级牌（逢人配）在进贡时会被系统自动跳过
- 进贡和还贡都有严格的时间限制，超时可能被判违规
