# 报牌规则

## 概述
掼蛋游戏中的报牌规则，包括报牌条件、报牌方式等。本文档依据江苏掼蛋规则（274-277行）的明确定义。

## 报牌规则（依据江苏掼蛋规则274-277行）

### 报牌条件
- **剩余牌数**：牌手在出完一手牌后，如手中剩余牌张为**10张(含)以内**，应及时、准确、清楚报出剩余张数
- **报牌时机**：在出完一手牌后，如果剩余牌数满足条件，应立即报牌
- **报牌内容**：报出剩余牌张的数量（如"还有5张"、"还有3张"等）

### 报牌方式
- **报牌方式**：及时、准确、清楚报出剩余张数
- **报牌器具**：正式比赛中可采用报牌器具报牌
- **禁止行为**：
  - 禁止牌手追问和反复报牌
  - 禁止在报牌后反复确认或追问

### 特殊情况
- **取消报牌**：在一定范围和阶段的比赛中可以取消报牌
- **平台处理**：在AI平台中，报牌可能由系统自动处理或取消

## 违规处理（依据江苏掼蛋规则）

### 警告
- **反复报牌、追问余牌**：给予警告
- **未报牌、报错牌，各家出牌(或放弃出牌)不超过一手牌**：予以收回，给予警告

### 违例
- **未报牌、报错牌，一家出牌(或放弃出牌)超过一手牌**：记违例
- **如一家在违规后出完牌(未获上游)**：予以收回，记违例

### 犯规
- **未报牌、报错牌，一家在违规后出完牌且获得上游**：判犯规
- **判罚结果**：本副牌结束，违规方双下，下副牌不进贡，由裁判在非违规方中指定首圈领出者

## 注意事项
- 报牌规则必须严格按照江苏掼蛋规则执行
- 剩余牌数10张(含)以内时必须报牌
- 报牌必须及时、准确、清楚
- 禁止反复报牌和追问余牌
- 未报牌或报错牌可能导致警告、违例或犯规
