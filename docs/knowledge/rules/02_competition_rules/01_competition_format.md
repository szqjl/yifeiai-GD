# 竞赛形式

## 概述
掼蛋比赛的竞赛形式和组织方式。

## 比赛形式
- **团体赛**：以队伍为单位参赛
- **个人赛**：以个人为单位参赛
- **混合赛**：团体和个人混合

## 比赛规则
- **局数**：根据比赛规则确定局数
- **计分方式**：按局计分，累计总分
- **胜负判定**：按总分或局数判定胜负

- [违规处理]

## 注意事项
- 比赛形式必须严格按照比赛规则执行
- 比赛过程中必须遵守赛风赛纪
