# 计分规则

## 概述
掼蛋游戏的计分规则，包括局分、场分等。

## 计分方式
- **局分**：每局根据出牌顺序计分
- **场分**：多局累计总分
- **胜负判定**：按总分或局数判定

## 注意事项
- 计分规则必须严格按照比赛规则执行
- 计分过程必须公正透明
