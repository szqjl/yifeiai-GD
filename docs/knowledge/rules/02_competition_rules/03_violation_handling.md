# 违规处理

## 概述
掼蛋比赛中的违规行为及其处理方式。

## 违规类型
- **出牌违规**：出牌不符合规则
- **时间违规**：超时出牌
- **行为违规**：违反赛风赛纪

## 处理方式
- **警告**：轻微违规给予警告
- **扣分**：严重违规扣除分数
- **取消资格**：严重违规取消比赛资格

## 相关规则
- [竞赛形式]
- [计分规则]

## 注意事项
- 违规处理必须严格按照比赛规则执行
- 违规处理决定应及时宣布
