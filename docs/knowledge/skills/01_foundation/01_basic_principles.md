# 掼蛋原则

## 概述

掼蛋游戏的核心原则体系：一个中心，两个基本点，三个有利，四项基本原则，五条高压线。这是掼蛋游戏的基础指导思想，是所有技巧和策略的根本遵循。

## 适用阶段

- **开局阶段**：✅ 完全适用
- **中局阶段**：✅ 完全适用
- **残局阶段**：✅ 完全适用

## 一、紧扣一个中心

**核心原则**：以争头游为中心。先争取头游，后送对家。

**平台变量名**：`myPos`（我的座位号），`curPos`（当前玩家座位号）

**说明**：
- 头游是掼蛋游戏的最高目标
- 在确保自己能够争头游的前提下，再考虑送对家
- 这是所有策略的出发点和归宿

## 二、把握两个基本点

### 1. 判断

**核心技能**：通过对手的首发出牌判断其牌力强弱。

#### 一是首发出单张 (`Single`)，是牌力强的信息

**平台变量名**：`Single`（单张）

**判断依据**：首发出单张，说明牌力强，单牌是优势轮次

#### 二是首发出小对 (`Pair`)，是牌力弱的信息

**平台变量名**：`Pair`（对子）

**判断依据**：首发出小对，说明牌力弱，对子可能是过渡牌型

#### 三是首发三张 (`Trips`)，也是示弱的信息

**平台变量名**：`Trips`（三张）

**判断依据**："前面三张，后面应有三带二"，后面要注意观察，合适送个三带二 (`ThreeWithTwo`) 给对家，而不要机械地再送三张

#### 四是开局小顺子 (`Straight`)，牌力是中等

**平台变量名**：`Straight`（顺子）

**判断依据**：开局小顺子，牌力是中等

#### 五是中性牌可开小三连（木板）对或小三连三（钢板）

**平台变量名**：
- `ThreePair`：三连对（木板）
- `TwoTrips`：钢板

**判断依据**：中性牌可开小三连对或小三连三

### 2. 记牌

**核心原则**：打牌不记牌，等于瞎胡来。

**记牌内容**：
- 记大牌
- 记炸弹 (`Bomb`)
- 记牌路
- 对手出的牌
- 对家出的牌
- 还要记关键张2、5、10等

## 三、明确三个有利

### 1. 组牌有利

**核心原则**：组牌是选手的基本功，抓到手的27张牌，进行优化组合，实现最佳排列，取得最大效益。

**五个要点**：

1. **最大限度地减少轮次**：
   - 组成三带二 (`ThreeWithTwo`)
   - 组成连牌 (`ThreePair`, `TwoTrips`)
   - 减少手数

2. **理出同花顺** (`StraightFlush`)：
   - 同花顺是强大的牌型，能压5张以内的炸弹

3. **组火牌力较弱的情况下，尽量组成小火大牌型；牌力很强的情况下，就要尽量组成大火，防止对手冲刺**

4. **最大限度地减少单张** (`Single`)

5. **机动灵活，留有变化，不把牌组死**

### 2. 炸牌有利

**核心原则**：合理使用炸弹 (`Bomb`)，炸点值大的牌。

**五个要点**：

1. **炸对手最后一个大王** (`R`)：
   - 对手发牌会先打自己的优势轮次，即单牌回王
   - 对手的第一个大王可以放一手
   - 第二个大王则应炸掉，此时单牌反而成为优势轮次

2. **炸对手的首对级牌**：
   - 级牌对子是对手转换牌路的关键牌型

3. **炸对手的首套顺子** (`Straight`)：
   - 对手起手顺子，往往还会有顺子回手

4. **炸对手的钢板** (`TwoTrips`)

5. **炸对手主攻手的第一个炸弹** (`Bomb`)

### 3. 进贡与回贡有利

**平台变量名**：
- `tribute`：进贡
- `back`：还贡

**进贡策略**：

1. **给对手进贡**：
   - 进附近同一花色多的牌
   - 留下同花五连张缺一的牌

2. **对家进贡**：
   - 进附近同一花色少的牌
   - 留下同花五连张缺一的牌

**还贡策略**：

3. **给对手还贡**：
   - "4、6不还5、7"
   - 拆牌拆三不拆二
   - 还3不还2
   - 不还孤张小牌
   - 还组成连牌后拆出的单牌
   - 还附近同一花色多的牌
   - 还进贡牌附近同一花色的牌
   - 还级牌附近的红心牌

4. **对家还贡**：
   - 还单不还双
   - "还2不还3"
   - 还牌5、7
   - 还附近同一花色少的牌
   - 尽量不要还级牌附近的红心牌

## 四、坚持四项基本原则

### 1. 坚持谁打谁收原则

**核心原理**："谁打谁收"是有效的进攻手段，先发的牌路最好有大牌可以收尾，这样一次可以减少2手。

**三个要点**：

1. **"谁打谁收"明确了岗位责任**：
   - 你在那个岗位上，你就得坚守，是谓各司其职
   - 本家主打的牌，本家当然应当回收，不给对家无谓的压力

2. **"谁打谁收"彰显了对门双方的信任关系**：
   - 要信任对手，他既然这样发，肯定有他的道理
   - 在掼蛋过程中，特别是开局阶段，主打方基本都选择打自己的优势牌
   - 在这样的客观情况下，主打人是具备回接能力的，不用担心对手方的阻拦

3. **对家先发的牌，你接牌要慎重**：
   - 避免影响对家回收

### 2. 坚持"配合至上"原则

**核心原理**：配合原则：战略统领战术，战术体现战略。

**四个要点**：

1. **让道**：
   - 对家出小单张 (`Single`)、小对子 (`Pair`)，尤其是小顺子 (`Straight`) 的出现，就算能套也最好不要，留顺子可以送回去
   - 牌好才接，除非过了这手牌就能争上游
   - 牌不好不但不接，上家出牌有时都不要，让对家接

2. **反向喂牌**：
   - 对手已经出了两种牌形，对家均不要，出炸夺得出牌权，出与对手牌形相反的牌
   - **案例**：如对手出小顺子，对家不要，可以判断对家手中牌以"三带二" (`ThreeWithTwo`) 为主，判断准后出炸或拆牌组成大顺子压后夺得主动权，出牌时可出小"三带二"让对家过牌

3. **拆牌喂牌**：
   - 就是明知对家需要什么牌形，你手上没有这种牌形，那也要想办法拆牌组成这个牌形也要喂给对家
   - 甚至拆炸组成对家需要的牌形，以保证对家争上游

4. **"先大后小"巧喂牌**：
   - 刚开始喂牌尽量出大一点牌，让下家阻击
   - 你在还击后再出稍大一点牌
   - 几次过后，消耗了下家的阻挡力后，出小牌下家因无力再挡

### 3. 坚持打上家卡下家原则

**平台变量名**：
- `curPos`：当前玩家座位号
- `myPos`：我的座位号
- 上家：`(myPos - 1) % 4`
- 下家：`(myPos + 1) % 4`

**打上家（盯上家）**：

1. 盯上家第一起手牌
2. 盯上家直接喊过的牌形
3. 盯上家中牌区
4. 盯上家的特殊牌形

**卡下家（控下）**：

1. 控下家的过手牌
2. 控下家的上手牌
3. 控下家的待传牌
4. 控下家的牌形

### 4. 坚持强牌弱打弱牌强打原则

**强牌弱打**：

**原理**：
- 一方面为了吸引并消灭对手选手主力
- 另一方面也是给对家展示牌力和牌型的机会

**策略**：
- 如果对家牌力也强，就一起努力获得上游和二游的双升局面
- 如果对家牌力一般，就适当支持和配合对家，避免对家被对手选手合力阻击成为下游

**弱牌强打**：

**场景**：牌力一般，下家出的牌型对家接不住，作为最后的防守方炸弹阻击。

**策略**：
- 然后送出对家出过的牌型或者对家需要的牌型
- 牺牲自己支持对家争上游
- 这时候火打第一夯（三带二 `ThreeWithTwo`）或者第一顺 (`Straight`)

## 五、五条高压线

### 1. 进贡慎出单

**平台变量名**：`tribute`（进贡），`Single`（单张）

**核心原则**：一方进贡，特别是双贡，起手就出单牌，这是掼蛋的大忌。

**原因**：因为自己的大牌已经贡出去了，若开始就打单牌，这就等于拱手让出自己的"发球权"

### 2. 火不打四

**平台变量名**：`Bomb`（炸弹）

**核心原则**：一般情况下，对手只剩四张牌的时候就不打了，将其放其走，因为4张牌极有可能是一炸。即使不是一炸，他一把也出不干净，所以炸掉也无意义。

**相关原则**：
- 三张慎出对
- 五张慎出单
- 九张慎出五
- 还有五必炸，六必治，七八让发

### 3. 尽量避免一打二

**核心原则**：尤其是打A，拦下一家就是赢。

**策略**：
- 除非牌特别好才一打二
- 防止炸了东家炸西家，不管什么牌，哪家的牌，能压就压
- 跟对家形不成配合，各打各的，甚至还相互抢牌，浪费牌力，给对手可乘之机

### 4. 不打赌气牌

**核心原则**：在掼蛋牌局中，经常发生这样的现象，打着打着，就和对手较上劲了，你出炸我马上还炸拚个你死我活，不是鱼死就是网破，都耗光了所有的炸弹。

**错误后果**：
- 这种打赌气牌是不可取的
- 逞个人主义，做拚命三郎，只会导致这局牌完败

**正确做法**：
- 特别是对手一方拥有超强牌力，尽量不要攻击对手的强牌选手
- 要避其锋芒，不要和拳王干架，要保存实力

### 5. 顺子慎始发

**平台变量名**：`Straight`（顺子）

**核心原则**：能先出牌的，一般都是进贡的一方，基本没有单牌的优势。如果开始就出顺子，对手很容易调整出大顺子来加以阻拦，而他们单下来的一些牌，又不愁走不掉，所以对自己是极为不利的。

### 6. 不要越级上小王

**平台变量名**：`B`（小王）

**核心原则**：不要越级上小王，应该按照牌点顺序出牌，避免浪费小王。

**策略要点**：
- 应该按照牌点顺序出牌，不要越级上小王
- 小王 (`B`) 是重要的封顶牌，不应该轻易浪费
- 如果越级上小王，可能会浪费小王，导致后续无法封顶

**注意事项**：
- 不要为了压制小牌而越级上小王
- 应该保留小王，用于关键时刻封顶

## 注意事项

1. **一个中心是根本**：所有策略都要围绕争头游这个中心
2. **两个基本点是基础**：判断和记牌是掼蛋的基本功
3. **三个有利是方法**：组牌、炸牌、进贡还贡都要有利
4. **四项基本原则是准则**：谁打谁收、配合至上、打上家卡下家、强牌弱打弱牌强打
5. **五条高压线是底线**：绝对不能违反
