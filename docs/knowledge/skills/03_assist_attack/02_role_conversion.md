# 主攻转助攻技巧

## 概述

掼蛋游戏中，角色定位不是一成不变的。根据牌局形势的变化，主攻可以转换为助攻，助攻也可以转换为主攻。**掼蛋就是这么神奇，在不同的情况下，助攻可转换成主攻，主攻也可以转换成助攻。**

**核心原理**：根据牌力变化、对家牌力、对手情况，灵活调整角色定位，从主攻转换为助攻，配合对家争取头游。

**大家一定要知道该如何做主攻**：在掼蛋中，主攻和助攻的角色定位非常重要。主攻需要争取头游，助攻需要配合对家。只有清楚地知道如何做主攻，才能更好地配合对家，实现团队胜利。这样你才知道怎么去配合对家，怎么去为对家去阻挡对手的牌力。

**平台变量名**：
- `main_rank`：级牌
- `Bomb`：炸弹
- `ThreePair`：三连对（木板）
- `ThreeWithTwo`：三带二（夯）
- `Pair`：对子
- `Single`：单张
- `tribute`：进贡
- `back`：还贡

## 适用阶段

- **开局阶段**：⚠️ 部分适用，主要用于角色定位
- **中局阶段**：✅ 主要适用阶段，角色转换的核心应用期
- **残局阶段**：✅ 高度适用，残局角色转换是决定胜负的关键

## 一、角色转换的时机判断

### （一）牌力评估

**核心原理**：通过牌力计算，判断当前牌力是否适合主攻角色。

**牌力分级**：
- **8分以上**：超强牌，定位主攻
- **5-7分**：冲刺头游的强牌，攻守兼备，可以做主攻
- **2-4分**：中弱牌力，定位助攻
- **1分以下**：超弱牌，定位未游

**特别说明**：
- **6分牌可以做主攻角色**：在5-7分范围内，6分牌属于攻守兼备的强牌，可以尝试做主攻
- **4-5分是主攻和助攻的临界点**：根据对家牌力来做最后定位

### （二）主攻转助攻的时机

**核心原理**：当牌力下降、对家牌力增强、或对手情况变化时，主攻应该转换为助攻。

**转换时机**：

1. **牌力下降**：
   - 开局时是6分牌，可以做主攻
   - 但出牌过程中，牌力消耗，大牌已出
   - 此时应该转换为助攻，配合对家

2. **对家牌力增强**：
   - 对家出牌显示牌力很强
   - 对家已经取得出牌权，连续出牌
   - 此时应该转换为助攻，配合对家

3. **对手情况变化**：
   - 对手双进贡，我方优势明显
   - 对手牌力消耗，对家有机会头游
   - 此时应该转换为助攻，配合对家

4. **残局阶段**：
   - 对家剩牌较少，有机会头游
   - 自己牌力不足以争头游
   - 此时应该转换为助攻，配合对家

5. **判断对家角色后转换**：
   - 观察对家出牌，判断对家也是主攻角色
   - 如果对家牌力很强，且已经取得出牌权
   - 此时应该转换为助攻，配合对家

6. **获得出牌权后的转换**：
   - 获得出牌权后，观察对家是否跟牌
   - 如果对家跟了一手三连对 (`ThreePair`) 或其他牌型，显示对家牌力强
   - 此时应该转换为助攻，配合对家

## 二、主攻转助攻的策略调整

### （一）组牌策略调整

**核心原理**：从主攻的"轮次优先"转为助攻的"配合优先"。

**策略变化**：

1. **主攻组牌**：
   - 轮次越少越好，可以拆炸弹 (`Bomb`)，减少轮次
   - 炸弹越多越好，一般不拆炸弹组同花顺 (`StraightFlush`)
   - 牌型尽量一致，保持连续进攻

2. **助攻组牌**：
   - 保留炸弹，必要时阻击对方
   - 炸弹越大越好，可以拆炸弹组大同花顺
   - 尽量保持牌型多样，关键时候送对家

**转换后的组牌策略**：

1. **保留炸弹**：
   - 不再为了减少轮次而拆炸弹
   - 保留炸弹，用于配合对家或阻击对手

2. **保留多种牌型**：
   - 不再追求牌型一致
   - 保留多种牌型（单张、对子、三带二、顺子），方便配合对家

3. **保留3连对和3带2的牌型**：
   - 保留一个三连对 (`ThreePair`) 和3带二 (`ThreeWithTwo`) 的牌型
   - 这样可以根据对家需要，灵活送牌

### （二）出牌策略调整

**核心原理**：从主攻的"主动出牌"转为助攻的"配合出牌"。

**策略变化**：

1. **主攻出牌**：
   - 主动出牌，争取出牌权
   - 连续出牌，快速跑牌
   - 使用大牌压制对手

2. **助攻出牌**：
   - 观察对家牌型，配合对家
   - 对家出牌时，尽量PASS，让对家继续
   - 对家需要时，及时送牌

**转换后的出牌策略**：

1. **让对家出牌**：
   - 对家出牌时，尽量PASS
   - 让对家连续出牌，争取头游
   - 对家打一张3 (`Single`)，如果对家牌力强，应该PASS让对家继续

2. **及时送牌**：
   - 判断对家牌型，及时送对家需要的牌
   - 送牌要精准，不要送错牌型
   - 对家跟了一手三连对 (`ThreePair`)，说明对家可能需要三连对或相关牌型

3. **阻击对手**：
   - 对手出牌时，如果对家不要，可以阻击
   - 使用炸弹阻击对手，保护对家

4. **判断出牌权机会**：
   - 观察是否有机会获得出牌权
   - 如果对家牌力强，即使有机会获得出牌权，也可以选择PASS
   - 如果对家需要配合，可以主动获得出牌权后送牌

5. **"那我们先不要"的决策策略**：
   - 当对手出牌时，如果对家牌力强，可以选择"先不要"（PASS）
   - 让对家处理，观察对家是否跟牌
   - 如果对家跟牌，说明对家牌力强，应该转换为助攻
   - 如果对家不跟，再考虑自己出牌

### （三）进贡策略调整

**核心原理**：进贡阶段的选择，影响后续角色定位。

**策略示例**：

1. **对手双进贡的情况**：
   - 对手双进贡，我方优势明显
   - 可以选择还贡一张小牌（如2）
   - 这样可以让对家获得更好的牌，增强对家牌力

2. **还贡选择**：
   - 如果准备转换为助攻，还贡时可以选择小牌
   - 让对家获得更好的牌，增强对家牌力
   - 例如："那我们选择还贡一张2"

## 三、主攻转助攻的组牌示例

### 示例：6分牌的主攻转助攻

**初始状态**：
- 牌力：6分（可以做主攻）
- 手牌：包含四炸（四个J）、五炸（五个4）等
- 定位：主攻

**组牌策略**：
- 可以组成三手火牌（炸弹）的牌型

**转换时机**：
- 出牌过程中，牌力消耗
- 对家牌力显示很强
- 对手双进贡，对家有机会头游

**转换后的策略**：
- 保留炸弹，用于配合对家或阻击对手
- 保留多种牌型，方便配合对家
- 对家出牌时，尽量PASS
- 对家需要时，及时送牌

## 四、主攻转助攻的注意事项

### ⚠️ 重要说明：真实情况的复杂性

**核心提醒**：本文档中的策略和案例主要来自视频教程演示，展示了主攻转助攻的典型场景。**真实掼蛋游戏中，情况千变万化，不能机械套用这些策略。**

**实际应用原则**：
1. **灵活应对**：根据实际牌局形势、对家表现、对手情况，灵活调整策略
2. **综合分析**：不要只看单一因素，要综合考虑牌力、牌型、时机、对手等多方面因素
3. **动态调整**：角色转换不是一次性的，可能需要多次调整
4. **因地制宜**：同样的牌型，在不同局面下可能需要不同的策略
5. **经验积累**：通过大量实战，积累经验，提高判断准确性

**视频演示的局限性**：
- 视频只能展示特定场景的演示
- 真实情况可能更复杂，需要更多判断
- 不能完全依赖视频中的固定模式
- 要根据实际情况灵活变通

### 1. 及时转换

**核心原理**：不要固执地坚持主攻角色，要根据牌局形势及时转换。

**错误做法**：
- 牌力已经下降，还坚持主攻角色
- 对家牌力很强，还坚持自己出牌
- 对手情况变化，还坚持原定策略

**正确做法**：
- 观察牌局形势，及时判断是否需要转换
- 对家牌力增强时，及时转换为助攻
- 对手情况变化时，灵活调整策略
- **真实情况千变万化，需要灵活应对，不能机械套用**

### 2. 配合对家

**核心原理**：转换为助攻后，要以配合对家为核心。

**策略要点**：
- 对家出牌时，尽量PASS
- 对家需要时，及时送牌
- 对手出牌时，如果对家不要，可以阻击

### 3. 保留灵活性

**核心原理**：组牌时保留多种牌型，保持灵活性。

**策略要点**：
- 保留一个三连对 (`ThreePair`) 和3带二 (`ThreeWithTwo`) 的牌型
- 保留多种牌型（单张、对子、三带二、顺子）
- 这样可以根据对家需要，灵活送牌

## 六、总结

**核心要点**：

1. **角色转换是灵活的**：主攻可以转换为助攻，助攻也可以转换为主攻

3. **转换后调整策略**：从主攻的"轮次优先"转为助攻的"配合优先"
4. **保留灵活性**：组牌时保留多种牌型，保持灵活性
5. **配合对家**：转换为助攻后，要以配合对家为核心

**关键原则**：
- **掼蛋就是这么神奇，在不同的情况下，助攻可转换成主攻，主攻也可以转换成助攻**
- **6分牌可以做主攻角色，但要根据牌局形势，及时转换为助攻**
- **保留一个三连对和3带2的牌型，方便配合对家**

**⚠️ 重要提醒**：
- **真实掼蛋游戏中，情况千变万化，不能机械套用这些策略**
- **要根据实际牌局形势、对家表现、对手情况，灵活调整策略**
