# 掼蛋牌语

## 概述

鸟有鸟语，人有人语，牌语就是牌手出牌、压牌或过牌让牌中明示或隐含的技巧和逻辑信息。心有灵犀一点通，牌手的沟通不是靠语言，也不是手势，而是靠牌语。牌手打出每一手牌，都含有一定信息，通过这些信息，告诉对家我在做什么、我的牌型如何、我的牌力强弱，以意图是什么，读懂牌语，识别牌语，就能最大限发挥自己打牌的技术水平。

**平台变量名**：
- `myPos`：我的座位号
- `curPos`：当前玩家座位号
- 上家：`(myPos - 1) % 4`
- 下家：`(myPos + 1) % 4`
- 对家：`(myPos + 2) % 4`

## 适用阶段

- **开局阶段**：✅ 完全适用，开局牌语最重要
- **中局阶段**：✅ 完全适用
- **残局阶段**：✅ 完全适用，残局牌语决定胜负

## 一、开局出牌信息，判断牌力

**核心原则**：读牌，首先要看懂自己手里牌，再读其余三家牌语。读懂自己手里的牌，主要是看大牌数量，牌型是否整齐，弱轮次的数量，炸弹的数量和质量，是否能获得出牌权。

**角色定位**：通过获得出牌权后首发牌或者首次顺牌传递信息，判断他想做什么样角色定位，从而调整自己定位。
- **主攻角色**：首次出牌或者是首次顺牌是为了处理自己弱路
- **助攻角色**：为了让对家顺牌或者送对家牌

### 1. 首发出小单牌 (`Single`)，是牌力强的信息

**平台变量名**：`Single`（单张）

**牌语含义**：牌强出单牌，对家一看就要明白对家牌力强的信息，就要配合让对家争头游。

**对家应对**：
1. **对家牌力也强**：就要考虑双上
2. **若对家牌力不强**：一定要牺牲自己、全力配合，根据对家出牌的信息，给对家送去他要想的牌型，力保对家抢出头游来

**对手应对**：
- 对手方高度警惕，要有意加强阻拦，变换牌型，抑其长而攻其短
- 尤其是首发牌者是双进贡上家，要尽力抬高、刻意扛起，不给对手过牌
- 末家上手后，一定不能出首发的同类型小牌

### 2. 开局出对 (`Pair`)

**平台变量名**：`Pair`（对子）

**牌语含义**：情况不明，对子先行。出于尝试沟通，了解对家牌力。

**判断方法**：

1. **看后续发牌牌力展示**：
   - 首发小对子，大对子回收，又出1对牌，表明对子占优势，对家就知道怎么送牌，对手也知道防守
   - 首发小对子，没有回收或者回收不在出对，表明牌力弱

2. **对家是否接牌牌力展示**：
   - 首发出对，不管下家对手要不要，对家接牌，说明对家牌力大概率中等偏上
   - 对家牌力不济或中等，一般不会主动压对家的牌，尤其在开局阶段

3. **对家应对**：
   - 对付对子起步，一般是中等对子防守，不建议卡压
   - 牌力强可以卡压，再亮出自己主打牌型

### 3. 开局出高单 (`Single`)

**牌语含义**：

1. **吃贡方防止下家过小牌**：出高单，然后回收
2. **牌力一般，定位助攻**：能出尽出，能走尽走。尽快让对家知道需要什么牌型，方便协助支持。对家是不是真正想传递这个信息，还得要看对手回不回接牌才能够进一步确认
3. **开局交牌权**：牌手上游，对家下游抗贡。首发高单，把控单权交给对家。一般情况下，高手会采取"单控控单"的原则，即明知单张控，开始不出单张，等着对手换轮次时主动出单，进而获得控牌权。一般也是向对家传递定位助攻的信号

### 4. 开局小顺子 (`Straight`)，牌力是中性

**平台变量名**：`Straight`（顺子）

**牌语含义**：开局出个小杂花顺或小三带二 (`ThreeWithTwo`)。

**两种情况**：

1. **"有打有收"**（回手有到K以上的牌），边打边看对手牌力：
   - 对家会根据自己牌力来决定是让过、顺过还是接牌
   - 对家牌力弱可让过，中性牌也可顺过，牌力强恰又有大牌，接管也行
   - 首发牌被对手阻断，对家合适时候给对家传递、喂送这种牌型

2. **牌力强，其它牌型比较大，想抢头游**：也可以开局出个小三带两或小杂花顺，而不用有回收牌，这叫作"小死牌不出没人给出"

### 5. 开局出木板 (`ThreePair`) 或钢板 (`TwoTrips`)，一般中性牌

**平台变量名**：
- `ThreePair`：三连对（木板）
- `TwoTrips`：钢板

**牌语含义**：开牌首发牌型概率相对小木板或钢板，想能再出一手牌，或者要出末家一个炸弹。

**判断**：
- 出木板一般三个头少，炸弹少
- 首出钢板一般对子少，炸弹一般也不多

**应对原则**："下家不拆牌，交给末家来"。

**策略**：
- 首发下家不要强拆牌或轻易出炸弹，要交给对家（即末家），看有没有现成的牌压，或者开炸
- 末家牌力强、炸弹多，可拆可炸，不再给首发者再出牌的机会
- 末家炸弹不多，也可以放过，看其出第二手牌的牌型
- 原则上首发者的第二手牌，末家一定要负责，管控或开炸，"末家负责制"
- 不要轻易给一个人连出三手牌机会

### 6. 开局出三张 (`Trips`)，一般是弱牌

**平台变量名**：`Trips`（三张）

**牌语含义**：首发小三张，向对家表示牌力弱。一般属于捣蛋牌、干扰牌，俗话说"要想坏，三张"，特别是发三张再出对子，表明牌力很差，送对家上手。

**判断**：
1. 开局出个小三张，一般手上有两个以上三个头、且对子较少或者对子较大
2. 对子大，就是故意引诱对手立牌后出对子
3. "出个三张，后面必有三个带"，后面还有三带二 (`ThreeWithTwo`)。对家要注意，合适时送个三带二给对家，而不要机械地再送三张

**应对**：
- 对付三张，第二家可以顺过、也可以不予理睬，牌力特别强时才可考虑卡压或炸掉
- 末家要根据自己的牌型和牌力来决定管控或炸掉第二手

### 7. 三带顺子少，出顺对子多

**核心原则**：首家发牌后，对家要注意，首家出三带二 (`ThreeWithTwo`)，原则上就顺子少；首家出顺子 (`Straight`)，原则上就对子多，行牌过程中要根据这个原则，给对家送牌、喂牌。

**策略**：
- 就是对家出三带二，对家上手后要注意送三带二，送不了就送单牌或对子，一定不要轻易出顺子
- 对家若出的是顺子，对家上手后就要送顺子，送不了顺子就出对子或单张，不要出顺子三带二

## 二、对家、对手方应对信息

**核心原则**：该懂其余三家的牌语，就是读懂在行牌过程中其他三位牌手所释放的信息。在出牌的时候会传递很多信息，对家档双方通过牌的信息，来全力配合、争取获胜，"你吃我喂，你短我断"——即对家需要牌型，要全力、及时递过去；对家没有牌型，属于短板，要全力管控、阻断对手，迅速为对家改变牌型。

### （一）上家领出牌

**平台变量名**：上家 = `(myPos - 1) % 4`

#### 1. 上家出小单张 (`Single`)

**牌语含义**：
- 跟出最小单张牌，表明牌力较强，需要处理一些难以出掉牌
- 出一张牌点较高单张牌，如K或A，来试探对手是否持有相应级牌或王牌，表明牌力较弱

#### 2. 上家出小对子 (`Pair`)

**牌语含义**：
- 上家出对子，有对子尽量跟牌，以避免暴露手中牌型弱点
- 跟对子，不需要出点数太大对子，以免限制对家出牌机会
- 对家出花色以上对子被压，意味着对家不想继续出对子，考虑出炸弹来掌握牌局主动权，并尝试引导其他牌型出牌
- 不跟对子，说明手中顺子多

#### 3. 顺子和三带二

**牌语含义**：上家首先出顺子 (`Straight`) 或三带二 (`ThreeWithTwo`)，这些牌型上有多手牌，并可能具备回手牌。

**应对**：
- 牌力较强，只有一手难以出掉相同牌型，可以选择跟牌
- 一般情况顺过，上手后再送同样牌型
- 对手应该考虑阻击，改出能够克制对手牌型，或者至少领出不同牌型

#### 4. 不接对手牌

**牌语含义**：
1. 没有这种牌型、或不是其优势牌型、或牌力不济牌力一般，正在尽可能保留各种牌型和变化，以便在残局阶段适时传牌、送牌
2. 判断上家牌力强而自己牌力稍弱，对家如牌力不是非常强，可以放一人打一人，以防我方被打成双下

#### 5. 炸弹炸下家 (`Bomb`)

**平台变量名**：`Bomb`（炸弹），下家 = `(myPos + 1) % 4`

**牌语含义**：是表示牌力强，想上游，体现强势牌，正常情况下是炸上家，控下家，送对家。也不排除配合对家，炸下家进行佯攻，吸引出对手的火力，保证对家顺利取得上游。

### （二）对家领出牌

**平台变量名**：对家 = `(myPos + 2) % 4`

#### 1. 不接对家牌

**牌语含义**：原则上对家主动出牌，即使接得上也让牌，除非有冲上游优势，否则会扰乱对家节奏和计划。

**策略**：
- 对家强牌强打，不要主动改对家牌型，悄悄助攻
- 对家弱牌强打，主动让牌，节约自己兵力，创造机会拯救对家

#### 2. 主动接对家牌想上游

**牌语含义**：体现是强势

#### 3. 判断对家是否有送牌

**牌语含义**：对家明显送牌，应该及时接牌，这通常发生在以下几种情况：
1. 出单张被阻击后，对家上手并出单张
2. 明大王 (`R`)，对家上手出单张
3. 出对子、三带二或顺子被阻击后，对家上手并出相应牌型

#### 4. 自己出顺 (`Straight`)，下家不要，上家也不要

**牌语含义**：
- 说明上家火力不富裕
- 自己再出顺，下家继续不要，说明下家炸弹也不富裕，起码是无小炸弹

## 三、牌型牌语

**核心原则**：牌型出牌顺序大小说明牌语。

### 1. 出牌先后信息

**核心原则**：按照谁打谁收原则，先出顺子 (`Straight`)，对手不要，接着出三带二 (`ThreeWithTwo`)，一般剩余顺子；先出三带二，对手不要，再出顺子，一般剩余三带二。

### 2. 三带二 (`ThreeWithTwo`)，带对子大小信息

**平台变量名**：`ThreeWithTwo`（三带二）

**牌语含义**：
1. **第一手带大对子，第二手带小对子**：说明还有三带二，希望对家传三带二
2. **第一手带小对子，第二手带大对子**：说明不让对家传三带二
3. **33三带二2或者222带33**：说明没有三带二或者不希望传三带二
4. **带对子KK或者AA**：说明不希望传三带二和对子
5. **出55544或10101088**：说明家里没顺子

### 3. 炸弹 (`Bomb`) 大小信息

**平台变量名**：`Bomb`（炸弹）

**牌语含义**：
- **相邻炸弹，第一炸大，第二炸小**：说明还有炸弹或者大炸弹，如第一手9999，第二手8888
- **相邻炸弹，第一炸小，第二炸大**：说明没有炸弹，第一手8888，第二手9999

### 4. 出牌反映牌语

**牌语含义**：
1. **出33399**：家里可能有4——8顺子 (`Straight`)
2. **出顺子则少三带二**：剩下是单张 (`Single`) 或对牌 (`Pair`)
3. **出三张 (`Trips`) 表明对子少，出三带二 (`ThreeWithTwo`) 说明顺子少**
4. **出4张同一花色的顺子**：说明家里无红心配，同花顺 (`StraightFlush`) 少。红心配在残局中基本上都是组同花顺

### 5. 需要对子 (`Pair`)

**判断方法**：
1. **主动出过顺子 (`Straight`) 后，又套了一手顺子**：则可能有对子，一定不会多太多的对子，最多一到两对，且对子不会太小，对子太多或太小，一定会先出对子
2. **上家出单 (`Single`)，对家拆对子套单**：如拆QQ、很可能也多对子，而且剩下对子比QQ大
3. **过8套K，又有8一Q的顺子**：这相当于变相拆对子，也是多对子
4. **套对手三带二 (`ThreeWithTwo`) (如44422) 时候，对家出JJJ33、AAA55之类大标志之一**：三个头带小对子牌，那么对家很可能有6-10顺子，且多对子

## 四、残局牌语

### 1. 突然停牌

**牌语含义**：初期强攻强打后突然停下来，不再跟牌压牌。

**可能原因**：
1. 已尽力，弹尽粮绝
2. 目前所出牌型不是需要的，正在等机会，必须要过掉一手牌，才能发起冲刺，需要配合支持一下
3. 已听牌

### 2. 残局时忽然出小单张 (`Single`)，一般是听牌标志

**牌语含义**：手上是净炸 (`Bomb`)，或一炸一手牌，随时可以冲刺。

**应对**：
- 对手要根据牌力，决定是强打还是放过
- 对家要主动接牌，尽快走牌，对家随时可出手中最后一炸，阻击对手强势牌型，可再次接手，获得出牌权

### 3. 净手炸接牌

**牌语含义**：空扔4个头炸 (`Bomb`)，报牌5张。这明显是个炸，对手一般不拦截。接着又丢出1单张 (`Single`)，剩余4张牌。

**应对**：如果对家有明大王 (`R`)，本家有红心配，最好出红心配，释放对家接牌信息，提醒队手接牌。

### 4. 残局从小出到大，对家一般不要轻易接牌

**牌语含义**：而对家从大出到小，还剩4——10张牌，表示已经听牌，对家考虑接牌。

### 5. 在对手已有一家上游，我方和对手成为二打一的情况下

**牌语含义**：对手上家有避免下游，两个单张 (`Single`)，出小牌留大牌，让对家好放过。

### 6. 让对家先走

**牌语含义**：对家剩4张净手炸 (`Bomb`)，上家出牌剩5——10张，有可能一同花 (`StraightFlush`) 加一手牌，自己不走，出牌不接，示意对家先走。

### 7. 要求垫炸

**牌语含义**：上家剩4张明显炸 (`Bomb`)，一般不会开炸，但是炸后出一手，剩6——10张，有可能一大炸加一手牌，示意对家垫炸抢头游。

## 注意事项

1. **牌语是一把双刃剑**：对家或对手牌语有可能是真实的，也有可能是伪装的
2. **实战中，不注意牌语是不可取的，一味关注牌语也没有必要**
3. **对家明显是配角辅助型打法**：根本不需要考虑对家真意图还是假动作，完全按照自己的节奏去打，不需要考虑对家的牌型
4. **根据牌面的情况和静态信息的分布，再根据动态牌的过程来研判对家和对手的牌型**：因地制宜，因势利导，做出有效的应对和选择
5. **这种判断经过一定量实践经验的积累，就会更加清晰明了，这便是所谓的"牌感"**
