# 掼蛋相生相克

## 概述

木生火，火生土，水克火，火克金，相生相克是古代阴阳五行学中一个重要理论基础，万事万物之间，不是相互滋生，相互促进，就是相互制约，相互克制。掼蛋中也蕴含相生相克的属性，顺子 (`Straight`) 和三带二 (`ThreeWithTwo`) 相克，三张 (`Trips`) 和对子 (`Pair`) 相克，三带二 (`ThreeWithTwo`) 一般情况下和对子 (`Pair`) 相克，顺子和对子既相生又相克。

**平台变量名**：
- `Straight`：顺子
- `ThreeWithTwo`：三带二
- `Trips`：三张（三张）
- `Pair`：对子
- `Single`：单张
- `Bomb`：炸弹

## 适用阶段

- **开局阶段**：✅ 适用，用于判断牌型
- **中局阶段**：✅ 主要适用阶段
- **残局阶段**：✅ 完全适用，残局判断的关键

## 一、单张、对子和顺子的关系

### 1. 组顺子通常会形成单和对子的局面

**核心原理**：组顺子 (`Straight`) 通常会形成单张 (`Single`) 和对子 (`Pair`) 的局面。

**判断方法**：

1. **牌手主动出过顺子后，又套了一手顺子**：
   - 则可能有对子 (`Pair`)
   - 但一定不会有太多的对子，最多一到两对，且对子不会太小
   - 对子太多或太小的话，一定会先出对子

2. **不要单张对子有顺**：
   - 本来组顺必有单张和对子，可到了中后期，后期完全不吃单和对子
   - 那么最有可能的牌型，就是顺子 (`Straight`)
   - 单张与对子却成了判断有没有顺子的依据，这就是典型的相生相克

### 2. 单牌少，一般会对子多；对子少，一般单牌多

**核心原理**：单牌 (`Single`) 和对子 (`Pair`) 之间存在互补关系。

**应用**：
- 如果对手单牌少，一般对子多
- 如果对手对子少，一般单牌多
- 可以根据这个规律来判断对手的牌型分布

### 3. 对子与三张相克

**核心原理**：对子 (`Pair`) 与三张 (`Trips`) 相克。

**判断方法**：

1. **对手一直在打对子**：则打三张 (`Trips`)

2. **对手首打三张**：
   - 要么对子少，要么对子大
   - 则打对子 (`Pair`)

3. **三张后边往往还有一个三带二**：
   - 出了两个以上三张后，一般会有三带二 (`ThreeWithTwo`)

4. **从没有打过单张，一直三带二或对子，几轮之后忽然不动**：
   - 很可能需要顺单张 (`Single`)

### 4. 拆对子预防

**判断方法**：下家出单 (`Single`)，牌手先不要后要，一般拆对子 (`Pair`)，如拆的是QQ，那么他很可能也多对子，而且剩下对子比QQ大。

## 二、炸与单张、三张的关系

### 1. 炸多，则三带二少

**核心原理**：炸弹 (`Bomb`) 多，则三带二 (`ThreeWithTwo`) 少。原因很简单，四个头、五个头或同花顺 (`StraightFlush`) 多，三带二自然就少。

**平台变量名**：
- `Bomb`：炸弹
- `ThreeWithTwo`：三带二
- `StraightFlush`：同花顺

### 2. 炸弹多则单张概率就高一些

**核心原理**：炸弹 (`Bomb`) 多则单张 (`Single`) 概率就高一些。

**应用**：可以根据对手炸弹数量来判断单张的概率。

### 3. 牌手总是出三带二或者三张，说明没有红心配，手中炸弹较少

**判断方法**：牌手总是出三带二 (`ThreeWithTwo`) 或者三张 (`Trips`)，说明没有红心配，手中炸弹 (`Bomb`) 较少。

### 4. 大牌少则牌型整

**核心原理**：大牌少则牌型整。

**应用**：吃贡方严防进贡没有级牌的对手，特别是进贡K的上家，千万不要让顺单张 (`Single`)，也许就一张单，顺过就是上游。

## 三、顺子与三带二的关系

**核心原理**：顺不要夯，夯不要顺，典型的相克牌型。

**平台变量名**：
- `Straight`：顺子
- `ThreeWithTwo`：三带二（夯）

### 1. 点位计算

**核心原理**：据顺子和三带二点位可预判对手手上的牌，出完顺子后，在顺子牌区内炸弹和三带二的概率低，反之单牌和对子的概率就高。

**判断方法**：

1. **出5——9的顺子 (`Straight`)**：
   - 大概率没有5——9的三张和炸弹 (`Bomb`)
   - 期间炸弹少（一般拆5头组顺子不拆4头组顺子）
   - 牌点在5——9有单牌 (`Single`) 或者对子 (`Pair`)

2. **出3个J带对9**：
   - 没有7到J，10到A的顺子 (`Straight`)

3. **出777带22**：
   - 说明手上没有小顺子 (`Straight`)

4. **出大顺子 (`Straight`)**：
   - 手中没有大的三带二 (`ThreeWithTwo`)，如10JQKA

5. **出最小三带二 (`ThreeWithTwo`)，又用顶牌接回对手的小三带二**：
   - 则这两个三带二的区间也可能是顺子 (`Straight`)
   - **案例**：比如出过333+44，用级牌AAA+66，则7——K区间很可能有顺子或者三连对 (`ThreePair`)

6. **出一个最小顺 (`Straight`)，又接了一把顶天顺**：
   - 则中间的牌或级牌也有可能是三带二 (`ThreeWithTwo`)
   - **案例**：比如出45678，和10JQKA两把顺子，顺子行程之外只有2、3和9没出来，这几张牌成三个头或炸弹 (`Bomb`) 的概率就较高

### 2. 顺子多，则三带二可能少

**核心原理**：
- 连续出了多个顺子 (`Straight`)，则一般不太可能有三带二 (`ThreeWithTwo`)
- 顺子多，对子 (`Pair`) 一般也会多，并可能有同花顺 (`StraightFlush`)
- 不要顺子，一般会有三带二、连对 (`ThreePair`) 或钢板 (`TwoTrips`)
- 连续出了多个三带二，则一般不太可能有顺子

**平台变量名**：
- `Straight`：顺子
- `ThreeWithTwo`：三带二
- `Pair`：对子
- `StraightFlush`：同花顺
- `ThreePair`：三连对
- `TwoTrips`：钢板

### 3. 出小顺被压没有接，手上三带二概率高

**判断方法**：出23456，被人家34567压住了接不回去，手上剩下的牌7以上的三个头概率很高，不要认为打了顺子 (`Straight`) 就不要三带二 (`ThreeWithTwo`)，一定要考察顺子的行程。

### 4. 三带二到顶

**判断方法**：牌手出过888AA，再有三带二 (`ThreeWithTwo`) 的可能性非常小，对子已经带到了AA，手中没有对子，对手牌张在10张以上，有顺子 (`Straight`) 概率就很大。

### 5. 传牌选择

**核心原则**：根据相生相克关系来选择传牌。

#### 一是对家三带二传牌

**策略**：对家一直在打三带二 (`ThreeWithTwo`)，没有三带二，牌又走不掉：
- **首选**：送三张 (`Trips`)
- **次选**：单张 (`Single`)
- **实在不行**：送对子 (`Pair`)
- **一般不要**：发顺子 (`Straight`)，尤其是有打无收的小顺子

#### 二是对家顺子传牌

**策略**：对家一直在打顺子 (`Straight`)，没有顺子：
- **首选**：送单张 (`Single`)
- **次选**：对子 (`Pair`)
- **别出**：三带二 (`ThreeWithTwo`)，尤其是有打无收的小三带二

#### 三是对手顺子出牌

**策略**：对手一直在打顺子 (`Straight`)，接手后想遏制对手：
- **首选**：三带二 (`ThreeWithTwo`)
- **其次**：对子 (`Pair`)
- **忌讳**：走单张 (`Single`)，尤其是小单张

### 6. 判断牌型

**核心原理**：根据听牌方式判断尾牌牌型。

**判断方法**：
- **主动听牌**：尾牌很可能是大的三带二 (`ThreeWithTwo`)
- **被动听牌时，出顺子 (`Straight`) 后听牌**：尾牌很可能是三带二
- **出三带二 (`ThreeWithTwo`) 后听牌**：尾牌很可能是顺子 (`Straight`)

## 四、相生相克反打

### 1. 利用相生相克的牌型来做套

**核心原理**：高手运用相生相克的道理诱骗，反其道而行之。

**案例**：
- 开局打小三带二 (`ThreeWithTwo`)，回收一个大三带二
- 对手炸掉后，会打什么牌形呢？大概率是顺子 (`Straight`)，轮次相克
- 那高手普遍喜欢抓这种思路，先交换三带二后，留下顺子在那里守株待兔
- 或者先交换顺子，诱使对手用大顺子封牌，然后等套小三带二

**判断方法**：就要用到那句精典牌经——**能打不能收，必有相克轮次**。
- 打出小三带二 (`ThreeWithTwo`) 却收不回来，手中大概率有顺子 (`Straight`)
- 打出小顺子 (`Straight`) 收不回来，手中大概率有三带二 (`ThreeWithTwo`)
- 这就是相生相克

### 2. 守株待兔

**核心原理**：遇到高手做套，有时候是无法避免中套的，发现对手老玩这招，也可以针对。

**应对手法**：
- 例如对手故意把三带二 (`ThreeWithTwo`) 拆开，用小三张来逼大三张 (`Trips`)，让手中多出小对子 (`Pair`)
- 以后遇到这种情况直接不要，等对手设套不成，改打对子时，再压过来打三带二，这样对手就尴尬

## 注意事项

1. **相生相克是判断牌型的重要依据**：通过观察对手出牌，利用相生相克关系来判断其牌型
2. **灵活运用**：既要利用相生相克关系来判断，也要防止对手利用相生相克关系来做套
3. **能打不能收，必有相克轮次**：这是判断对手牌型的重要原则
4. **既生瑜又生亮，相生相克，相互制约**：要明辨判断，灵活掌握
