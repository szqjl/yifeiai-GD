# 掼蛋算牌

## 概述

记得住牌张或牌型，并籍此计算牌，绝大多数新手是个头疼的问题。掼蛋，因其多变的特点，本质上是个大概率事件的动态演绎过程。算牌也就是通过已知牌的信息进行逻辑推理来判断别人手上还剩什么牌。

**平台变量名**：
- `handcards`：手牌
- `remain_cards`：剩余牌
- `history`：历史出牌记录

## 适用阶段

- **开局阶段**：✅ 适用，建立算牌基础
- **中局阶段**：✅ 主要适用阶段
- **残局阶段**：✅ 完全适用，残局算牌决定胜负

## 一、全手牌算牌

**核心原理**：每位玩家手中都有27张牌，除大王 (`R`)、小王 (`B`) 之外，每个牌点的牌都是8张，在抓完牌之后，查看自己的手牌，就可估算某个牌点的牌在外面还有几张。

**平台变量名**：
- `R`：大王
- `B`：小王

**重要性**：在组牌阶段就开始建立记牌和算牌的意识，培养记牌和算牌的习惯，训练记牌和算牌的能力，将会最大化减少失误率，对整个牌局发展的预见性、前瞻性和把控性将会大大增加。

### 1. 对外面牌型的记算

**核心原理**：算大牌是指算牌点在10以上的牌，根据自己手中牌点在10以上牌点数量，判断其他三家有几张牌点在10以上的牌。

**应用**：通过计算大牌分布，判断对手牌力强弱。

### 2. 通过缺牌算同牌炸

**核心原理**：断张较多，该断张在外面形成头炸 (`Bomb`) 的可能性就较大。

**平台变量名**：`Bomb`（炸弹）

**炸弹种类一：四大天王有无**

**炸弹种类二：六个头及以上炸弹有无可能**

**判断方法**：根据手中缺张，孤张，手中无红配牌，判断有无6头以上炸弹，重点关注10以下小张。

**炸弹种类三：存在同花顺花色可能性**

**判断方法**：
1. **看自己同花顺大小**：以此为基础，如自我方910JQK，看缺哪门花色A，缺哪门再看10JQK花色，直到出牌看到为止，如果这门花色一直没有出现，很可能就成同花顺 (`StraightFlush`)，910JQK就不看大
2. **关注红桃配是否出现**

**平台变量名**：`StraightFlush`（同花顺）

### 3. 通过进贡大小算牌

**核心原理**：进贡没有级牌，一般炸弹 (`Bomb`) 多，牌型整，是最可怕的隐藏对手，千万记住不能让他顺走小单 (`Single`)。有时就一手小单过去就可以头游。

**平台变量名**：
- `Bomb`：炸弹
- `Single`：单张
- `tribute`：进贡

### 3.1 通过对手出牌算剩余牌

**核心原理**：根据对手出牌，计算剩余牌的数量和分布。

**算牌示例**：
- **场景**：下家压了3个A带对2（AAA22）
- **计算**：那外面还剩3个A和6个2
- **应用**：根据剩余牌的数量，判断对手牌力和可能的牌型组合

**算牌要点**：
- 记录每个玩家出过的牌
- 计算每个牌点的剩余数量
- 根据剩余牌判断对手可能的牌型

**核心原理**：进贡没有级牌，一般炸弹 (`Bomb`) 多，牌型整，是最可怕的隐藏对手，千万记住不能让他顺走小单 (`Single`)。有时就一手小单过去就可以头游。

**平台变量名**：
- `tribute`：进贡
- `main_rank`：级牌
- `Bomb`：炸弹
- `Single`：单张

### 4. 通过概率算分布

**核心原理**：先看其他人出的大牌，再根据自己手上的大牌，用概率来判断剩下大牌的分布。

**概率分布规律**：
- **8张牌**：3、5分布，4、4分布的可能性最大
- **7张牌**：3、4分布的可能性最大
- **6张牌**：3、3分布与2、4分布的可能性最大
- **5张牌**：2、3分布的可能性最大

**应用**：用这个概率在牌局初期对全局牌作预判，在中后期的牌局中使用排除法就能更准确地记住这些大牌。

## 二、通过已出牌算牌

**核心原理**：已出牌分为两种：一是进贡和还贡的牌；二是各位玩家打出的牌。

### 1. 通过贡牌算大小王

**核心原理**：在出牌之前，末游需向头游进贡，在末游向头游进贡后，就可大概判断大王 (`R`) 和小王 (`B`) 在谁手中。

**平台变量名**：
- `tribute`：进贡
- `back`：还贡
- `R`：大王
- `B`：小王

**判断方法**：
1. **双进贡都是级牌**：4个王都在吃贡方
2. **单贡情况下，自己为三游且手中没有大王和小王**：末游向头游进贡一张大王，那么两张大王只能在头游和二游手中，两张小王在头游、二游和末游手中
3. **用一对大王压对子 (`Pair`)**：一般手中还会有张小王
4. **先出大王，后出小王**：一般是对小王

### 2. 双贡一方首发小单牌

**牌语含义**：双贡一方首发小单牌 (`Single`)，牌应该不错，炸弹 (`Bomb`) 比较多，至少有上游的想法。

### 3. 通过已出牌排除同牌炸

**核心原理**：4张相同牌点的牌便可组成同牌炸 (`Bomb`)，当某个牌点的牌出现了5张后，便可排除这个牌点的同牌炸，除非用红心级牌配牌。

**案例**：例如，手牌中有JJ，上家打出JJJ-33，就可以排除由J组成同牌炸存在。

### 4. 顺子5和10定律

**核心原理**：除去红心配，5出尽就不会有小顺子 (`Straight`)，10出尽就不会有大顺子，记住场面上5和10的出牌张数，就可以判断牌局中顺子和同花顺 (`StraightFlush`) 的情况。

**应用**：如果牌局中10已经出完了，手里的小顺子，小同花照样做大登基。

**平台变量名**：
- `Straight`：顺子
- `StraightFlush`：同花顺

### 5. 孤张定律

**核心原理**：始终不见一张级牌 (`main_rank`) 身影，外面有含级牌的顺子 (`Straight`)，这就是"孤张定律"。

**判断方法**：具体是顺子还是同花，又是在谁手里，需要根据其他信息来辅助判断。"孤张定律"比"五十定律"在判断顺子方面，更好用一些。

**平台变量名**：
- `main_rank`：级牌
- `Straight`：顺子

### 6. 通过已出牌排除同牌炸

**核心原理**：算级牌时需注意将红心级牌与其他级牌区分开，不仅要算未出的级牌数量，还要算未出的红心级牌数量。

**案例**：例如，级牌为2，手牌中有红桃2和黑桃2，对家已出一对梅花2，上家用红心级牌配1手同花顺 (`StraightFlush`)，其余玩家手中级牌为两张方块2和一张黑桃2。

## 三、通过余牌算牌

**核心原理**：少于10张时就需要报牌，只估算10张以下的余牌。掼蛋中1手牌最多可出6张，4张牌是炸弹的分水岭，5张牌是顺子和三带二的分水岭，6张牌是三连对和三顺的分水岭。

**报牌张数的危险性判断**：
- **报牌7张是很危险的报牌张数**：报牌7张意味着可能还有一手牌（如三带二或顺子），需要特别警惕
- **下家直接开火报牌7张**：下家直接开火报牌7张，说明下家牌力很强，需要及时阻挡
- **分析对家手上的炸弹大小**：如果对家手上的炸弹 (`Bomb`) 比较大，可以配合对家，让对家处理

**平台变量名**：
- `Bomb`：炸弹
- `Straight`：顺子
- `ThreeWithTwo`：三带二
- `ThreePair`：三连对
- `TwoTrips`：钢板

### （一）剩余1-4张

**核心原理**：依据掼蛋可出牌型的组牌方式，4张牌以内的牌型有单牌 (`Single`)、对子 (`Pair`)、三张 (`Trips`)、四头炸 (`Bomb`)，可排除三带二 (`ThreeWithTwo`) 和连牌（顺子 (`Straight`)、三连对 (`ThreePair`)、三顺 (`TwoTrips`)）。

**算牌方法**：余牌在1——4张时，便可依照以上列出的牌型进行估算。

### （二）剩余5张

**核心原理**：依据掼蛋可出牌型的组牌方式，5张牌的牌型有三带二 (`ThreeWithTwo`)、顺子 (`Straight`)、同花顺 (`StraightFlush`)、五头炸 (`Bomb`)，或单牌 (`Single`)、对子 (`Pair`)、三张 (`Trips`)、四头炸 (`Bomb`) 的组合。

**算牌方法**：余牌为5张时，同花顺或者5头炸不做计算，一般以余牌为2手牌及以下的组合方式算牌，可猜测其余牌为顺子、三带二、四头炸加单牌这三种情况。

#### 1. 根据牌路判断

**判断方法**：之前出的对子 (`Pair`) 多，尾牌很可能是顺子 (`Straight`)；之前不吃顺子，尾牌很可能是三带二 (`ThreeWithTwo`)，反之亦然。

#### 2. 根据听牌主动性判断

**判断方法**：
1. **主动听牌**：尾牌很可能是大的三带二 (`ThreeWithTwo`)
2. **被动听牌时，出顺子 (`Straight`) 后听牌**：尾牌很可能是三带二
3. **出三带二 (`ThreeWithTwo`) 后被动听牌**：尾牌很可能是顺子 (`Straight`)

#### 3. 小炸+1手尾牌听牌

**判断方法**：小炸 (`Bomb`) 是4张牌，加上尾牌一般剩5张。

### （三）余牌为6——10张

**核心原理**：按照余牌为2手牌的方式算牌，可猜测其余牌为顺子加单牌、三带二加单牌、四头炸加对子、同花顺加单牌这四种情况。

**主攻出现最大概率**：
- **6张牌**：同花顺 (`StraightFlush`) + 单张 (`Single`)，4头炸 (`Bomb`) + 对子 (`Pair`)
- **7张牌**：同花顺 (`StraightFlush`) + 对子 (`Pair`)
- **8张牌**：同花顺 (`StraightFlush`) + 对子 (`Pair`) + 单张 (`Single`)
- **9张牌**：4头炸 (`Bomb`) + 顺子 (`Straight`)（夯 `ThreeWithTwo`）
- **10张牌**：同花顺 (`StraightFlush`) + 夯（顺子 `Straight`）

## 四、通过牌型相生相克属性算牌

**核心原理**：牌型有相生相克的属性，通过出牌情况，推算其手牌大致牌型。一般顺子 (`Straight`) 和三带二 (`ThreeWithTwo`) 相克；三张 (`Trips`) 和对子 (`Pair`) 相克；三带二 (`ThreeWithTwo`) 一般情况下和对子 (`Pair`) 相克；顺子和对子既相生又相克。

### 1. 点位计算

**核心原理**：据顺子和三带二点位可预判对手手上的牌，出完顺子后，在顺子牌区内炸弹和三带二的概率低，反之单牌和对子的概率就高。

**判断方法**：
1. **出5——9的顺子 (`Straight`)**：大概率没有5——9的三张和炸弹 (`Bomb`)，期间炸弹少（一般拆5头组顺子不拆4头组顺子），牌点在5——9有单牌 (`Single`) 或者对子 (`Pair`)
2. **出3个J带对9**：没有7到J，10到A的顺子 (`Straight`)
3. **出777带22**：说明手上没有小顺子 (`Straight`)
4. **出大顺子 (`Straight`)**：手中没有大的三带二 (`ThreeWithTwo`)，如10JQKA
5. **出最小三带二 (`ThreeWithTwo`)，又用顶牌接回对手的小三带二**：则这两个三带二的区间也可能是顺子 (`Straight`)。比如出过333+44，用级牌AAA+66，则7——K区间很可能有顺子或者三连对 (`ThreePair`)
6. **出一个最小顺 (`Straight`)，又接了一把顶天顺**：则中间的牌或级牌也有可能是三带二 (`ThreeWithTwo`)。比如出45678，和10JQKA两把顺子，顺子行程之外只有2、3和9没出来，这几张牌成三个头或炸弹 (`Bomb`) 的概率就较高

### 2. 顺子多，则三带二可能少

**核心原理**：
- 连续出了多个顺子 (`Straight`)，则一般不太可能有三带二 (`ThreeWithTwo`)
- 顺子多，对子 (`Pair`) 一般也会多，并可能有同花顺 (`StraightFlush`)
- 不要顺子，一般会有三带二、连对 (`ThreePair`) 或钢板 (`TwoTrips`)
- 连续出了多个三带二，则一般不太可能有顺子

### 3. 出了顺子又接了顺子

**判断方法**：
- 出了顺子 (`Straight`) 又接了顺子，则多半需要单张 (`Single`) 和对子 (`Pair`)
- 从没有打过单张，一直三带二 (`ThreeWithTwo`) 或对子 (`Pair`)，几轮之后忽然不动，很可能需要顺单张
- 出小顺被压没有接，手上三带二概率高。出23456，被人家34567压住了接不回去，手上剩下的牌7以上的三个头概率很高，不要认为打了顺子就不要三带两，一定要考察顺子的行程

### 4. 对手总是出三带二或者三张

**判断方法**：对手总是出三带二 (`ThreeWithTwo`) 或者三张 (`Trips`)，说明对手没有红心配，手中炸弹 (`Bomb`) 较少。

### 5. 特殊牌张5和10

**核心原理**：它们处于牌组的中间位置，组顺子时基本上都避不开这两张牌，不管是23456，或是10JQKA。

**判断方法**：有人打出101010+55或555+1010，那么他可能就没顺子 (`Straight`)。

### 6. 炸弹多，则单牌多的概率高

**核心原理**：炸弹 (`Bomb`) 多，则单牌 (`Single`) 多的概率高。

### 7. 没有单牌，一般会有对子

**核心原理**：
- 没有单牌 (`Single`)，一般会有对子 (`Pair`)
- 没有对子，一般会有单牌

### 8. 对子和顺子就是典型相生相克

**核心原理**：
- 本来组顺必有单张和对子，可到了中后期，单张与对子却成了判断有没有顺子的依据，这就是典型的相生相克
- 一般有顺必有对，一家抱着一把牌却始终不要对子，小对子他也没法套，手里肯定是顺子 (`Straight`)
- 牌局后期完全不吃单，那么最有可能的牌型，就是顺子

### 9. 相生相克反打

**核心原理**：打三带二 (`ThreeWithTwo`) 骗顺子 (`Straight`)，或打顺子骗三带二，这是当今掼蛋牌局中运用非常普遍的技巧。

**应用**：
- 高手也喜欢打单打吊对子，打对子吊单张，这些都是运用相生相克的道理在诱使对手出自己想要的轮次，是典型的反其道而行之
- **案例**：例如开局打小三带二，回收一个大三带二，对手炸掉后，会打什么牌形呢？大概率是顺子，轮次相克。那高手普遍喜欢抓这种思路，先交换三带二后，留下顺子在那里守株待兔，或者先交换顺子，诱使对手用大顺子封牌，然后等套小三带二

**判断方法**：就要用到那句精典牌经——**能打不能收，必有相克轮次**。
- 打出小三带二 (`ThreeWithTwo`) 却收不回来，手中大概率有顺子 (`Straight`)
- 打出小顺子 (`Straight`) 收不回来，手中大概率有三带二 (`ThreeWithTwo`)
- 这就是相生相克

## 算牌应用

**算对家牌型**：是为了适时传牌，力助对家。

**算下家对手牌型**：是为了防止在传对家，误放了对手。

**核心方法**：通过以上因"需"动态记牌，再积极运用"排除法"，随着牌程的推进，就可以进一步核实确认上述各种情况大概率存在的可能性。

## 注意事项

1. **会记算牌，未必能保证一定成为高手**：保证你每次都能获胜，但有一点却是确定的，不会记算牌，不仅肯定成不了高手，而且还会发生很多本不该发生的失误
2. **常输是大概率事件**：不会算牌，偶尔赢则变成了小概率事件
3. **看他没出过什么牌往往比看他出过什么牌更为重要**：比如有人出了45678，和10JQKA两把顺子，组顺的牌张之外只有2、3和9没出来，这几张牌成三个头或炸弹的概率就较高
4. **千万不要认为打了顺子就不要三带二**：如果一个人出了23456，被人家34567压住了接不回去，那他手上剩下的牌7以上的三个头概率很高
