# 掼蛋记牌

## 概述

打牌不记牌，等于瞎胡来。两副牌108张，每个人27张牌，由7种牌型组成，通过记牌，判断剩余牌的牌型和结构。理论上肯定是记得越多越好，但这会增加很多心智负担，尤其对于新手而言不如关注其他更重要的信息。

**平台变量名**：
- `handcards`：手牌
- `remain_cards`：剩余牌
- `history`：历史出牌记录

## 适用阶段

- **开局阶段**：✅ 适用，建立记牌基础
- **中局阶段**：✅ 主要适用阶段
- **残局阶段**：✅ 完全适用，残局记牌决定胜负

## 一、记大牌

**核心原理**：记大牌了解自己牌力变化，判断别人相应牌路强弱。最大的五个牌点：大王 (`R`)、小王 (`B`)、级牌 (`main_rank`)、A、K。大牌能影响胜负结果，大牌登基，意味着动态牌力的增长。

**平台变量名**：
- `R`：大王
- `B`：小王
- `main_rank`：级牌

### 1. 入门要求：记4张王

**核心原理**：
- 1张大王 (`R`) 出过，对小王 (`B`) 已登基，可以用来闯关
- 2张大王出过，单张小王已登基
- 4张王出过，级牌 (`main_rank`) 已登基
- 大小王各出1张，对级牌已登基

### 2. 进阶要求：记4张王+8张级牌

**核心原理**：同理，可以掌握A的登基情况。

**特别情况**：还与留风有关。

### 3. 精通要求：记4张王+8张级牌+8张A+8张K

**核心原理**：除了掌握Q的登基情况，这样还有利于推断全类型的大牌，和其它牌相结合，可以说99%的情况下，都够用了。

### 4. 掼神要求：记住所有10以上的大牌

**说明**：这个太难了，需要专业训练。

## 二、记小牌

**核心原理**：最小的五个牌点。打2时，34567是小牌。五个小牌点，并不需要全都记，挑重要的记就可以，主要记忆自己缺张和低于2张的牌点。到了残局阶段，往往手里会有小牌。原因有：一、小牌跟出去的机会少；二、有意不出，以控制下家。来记小牌避免过度防守，损伤牌力，能更有效地控制对手。

### 1. 入门要求

**要求**：没有要求，有观察小牌意识足够。

### 2. 进阶要求

**要求**：
1. **记住最小的8张牌**：勉强可以推断出全类型
2. **残局阶段，某小张未出大炸概率高**：如234出的很少，高度警惕对手手中可能存在五炸 (`Bomb`)、六炸等高牌力牌型

**平台变量名**：`Bomb`（炸弹）

### 3. 精通要求

**要求**：记最小16张牌，基本可以推断出全类型的小牌。

### 4. 惯神要求

**要求**：记住所有小牌，这个同样太难。

## 三、记炸弹

**核心原理**：炸弹 (`Bomb`) 的威力巨大，可以用来及时控制局势。推定对手已无炸弹，冲刺就相对容易多。

**平台变量名**：`Bomb`（炸弹）

**记炸弹的作用**：
1. 推定外面还有没有炸弹，这是冲刺阶段制定合理战术的前提
2. 推定头炸的余牌在谁家

### 1. 入门要求

**要求**：某家是否出过炸弹 (`Bomb`)。没出过炸弹，很可能还有炸弹。后期还没出炸，则一般至多有一炸。

### 2. 进阶要求

**要求**：出过的最大炸弹 (`Bomb`)。

**判断方法**：
1. **起步出5头炸**：那4头炸就不大可能有，后期同花炸 (`StraightFlush`)，可能是最后一炸
2. **出10JQKA同花炸**：则还有炸弹的很小
3. **出了6头炸**：几乎没炸。这从侧面说明："有大炸沉住气"，不要暴露太早

**平台变量名**：`StraightFlush`（同花顺）

### 3. 精通要求

**要求**：

1. **注意出牌频率**：当某一张牌被多次打出时，可以推算出含有该牌同花顺 (`StraightFlush`) 或连牌可能性降低。例如，红桃10连续被打出两次，那么红桃10——A同花顺就不太可能存在

2. **逢人配出过张数，出过所有炸弹及时间**：逢人配常用来配炸弹，还没出尽，外面很可能还有炸弹 (`Bomb`)

3. **成头炸，再成3同牌的可能就不大**：比如出过4张A，AAA三带二 (`ThreeWithTwo`) 可能性就很小

**平台变量名**：`ThreeWithTwo`（三带二）

4. **头炸本身和附近常有赘牌，注意卡位**

5. **很早就出炸，肯定还有炸**

### 4. 掼神要求

**要求**：推断头炸余牌在谁家。

**案例**：有4张9，对家出过单9，上家出过对9，则剩余一张9很可能在下家，不组顺子 (`Straight`) 就是单张 (`Single`)，注意防控。

**说明**：记住所有头炸，留意其余牌并作出相应判断，无疑就是大神。记大牌和小牌务求精准，记炸弹没有这么高的要求。

**平台变量名**：
- `Straight`：顺子
- `Single`：单张

## 四、记牌路

**核心原理**：牌路的作用：了解其他人的牌力和牌路强弱；推断余牌组成。

### 1. 入门要求

**要求**：记住先发牌路，先发什么牌路往往表明了他的牌力和用途。

**判断方法**：

1. **发小单张 (`Single`) 且不控下家**：一般表明牌力强。大牌少连续发单张，不是炸弹 (`Bomb`) 多就是有巨炸

2. **发对子 (`Pair`) 之后不接手**：一股表明牌力不强，起侦察作用。连发对子，对子就是强路，牌力不一定弱

3. **先发三带二 (`ThreeWithTwo`) 和顺子 (`Straight`)**：往往有接手牌

4. **先发木板 (`ThreePair`) 和钢板 (`TwoTrips`)**：一般是期望对手不开炸

5. **对手的意图要阻断，对家的意图被阻断后要助力接续**

**平台变量名**：
- `Single`：单张
- `Bomb`：炸弹
- `Pair`：对子
- `ThreeWithTwo`：三带二
- `Straight`：顺子
- `ThreePair`：三连对（木板）
- `TwoTrips`：钢板

### 2. 进阶要求

**要求**：注意三带二 (`ThreeWithTwo`) 牌路。

**判断方法**：
1. **判断余牌**：比如，对家出33399，上家用99944盖，你有一对9，则另一张9很可能在下家
2. **带对K**：则手中的对子 (`Pair`) 不多，对子是其短路
3. **出了级牌222**：则对子可能没有登基牌

### 3. 精通要求

**要求**：

1. **记住某家没出过的牌路和弱路**：
   - 从没出过单张 (`Single`)，很可能还有单张
   - 之前不吃对子 (`Pair`)，很可能手中没有对子

2. **跟牌，跟的是多大的牌**：特别要注意单张和对子的跟牌点位。
   - 下家剩2张，你先出单张不算什么，敢出对子才是高手
   - 下家余5张，你先出对子不算什么，敢出三带二 (`ThreeWithTwo`) 和顺子 (`Straight`) 才是高手
   - 对手登基后报5张，你开炸 (`Bomb`) 不算什么，放他出牌还打他下游才是高手

### 4. 掼神要求

**要求**：记住所有轮次出牌及过程，一副牌结束后可以准确地复述。

## 五、记关键张

### 1. 5和10是顺子关键张

**核心原理**：顺子 (`Straight`) 中必含5或10。8张10出完，则56789顺子很可能已经登基，除非逢人配组顺子。

**说明**：关键张可以和大牌一起记，但难度很大，也属于神级要求。

**平台变量名**：`Straight`（顺子）

### 2. 记缺张和单张

**核心原理**：
- 手中缺张，外面可能成炸 (`Bomb`)
- 手中单张 (`Single`)，外面可能有三带二 (`ThreeWithTwo`) 注意观察其分布情况

**说明**：记缺张和单张可作为进阶要求。

**平台变量名**：
- `Bomb`：炸弹
- `Single`：单张
- `ThreeWithTwo`：三带二

### 3. 记进贡回贡牌

**核心原理**：
- **对于胜方**：掌握进贡方最大牌，还要观察回贡牌使用，以提升回贡技巧
- **对于进贡方**：应尽快消除回贡牌这个记牌目标。比如对手回贡黑桃7，本身有红桃7，先发45678顺子 (`Straight`) 优先使用黑桃7

**说明**：记进贡回贡牌较为容易，可作为入门级要求。

**平台变量名**：
- `tribute`：进贡
- `back`：还贡
- `Straight`：顺子

### 4. 影响回手的牌

**判断方法**：比如有3个K，上下家都出过A或者出过6个A，大概率你就是台面最大的。

## 六、记牌方法

### 计数法："2468-"重点记牌法

**核心原理**：读作"2468 减"记牌法，这个记牌法就是要求能记住20张重点大牌。

**方法**：
- "2"是指两张红桃级牌 (`main_rank`)
- "4"是指四张王 (`R`, `B`)
- "6"是指六张级牌 (`main_rank`)
- "8"是指八张A
- "-"代表减号，是指要运用减法不断进行累计，减到0为止

**操作**：在心中各设一个计数，每出一张牌相应的计数减1，有点珠算味道。

**平台变量名**：
- `main_rank`：级牌
- `R`：大王
- `B`：小王

#### 1. 计数法优点

**优点**：
1. **只需要记住4个数**：四张王出完，只需要住3个数
2. **大牌种类一次不会出太多**：一般只有1个计数需要更新，很少出现222AA、A2345等需要同时更新两个计数的情况，比较容易掌握
3. **应用范围广泛**：既适合线上使用，也适合线下使用

#### 2. 计数法的缺点

**缺点**：
1. **需要一直在心里默念计数**：以防遗忘，分散注意力，多会影响对牌情的观察
2. **需要做减法计算**：多少耗费一些脑力

**说明**：计数法胜在简单，一学就会。如果把目标牌减少到1——2，尽可能地抑制缺点，那计数法仍不失为一个非常好的记牌方法。

## 注意事项

1. **记牌方法多种多样，适合自己最好**：记牌数量不是越多越好，不能为记牌而记牌
2. **随着记牌数量增加，记牌实际收益有递减趋势**：
   - 比如大牌记到A，90%情况下就够用
   - 多记一张到K，95%情况下就够用，收益增加5%
   - 多记一张到Q，96%情况下就够用，但收益只增加了1%
3. **记牌是为了"知已知彼"**：经过进一步计算和推断，作出合理决策
4. **要避免"为记牌而记牌"现象**：大多数情报并不左右战斗胜负，大多数记牌同样无关掼蛋输赢
