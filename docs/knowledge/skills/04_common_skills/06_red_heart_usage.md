# 红桃配运用

## 概述

在掼蛋的世界中，红心级牌（红桃配、逢人配）就像是一把多功能的瑞士军刀，它的用途广泛且强大，能够在关键时刻成为赛局的转折点。

**平台变量名**：
- `main_rank`：级牌
- `H` + `main_rank`：红心级牌（红桃配、逢人配）
- `Bomb`：炸弹
- `StraightFlush`：同花顺
- `Straight`：顺子
- `ThreePair`：三连对（木板）
- `TwoTrips`：钢板
- `ThreeWithTwo`：三带二
- `Pair`：对子

## 适用阶段

- **开局阶段**：✅ 适用，配牌组牌
- **中局阶段**：✅ 主要适用阶段
- **残局阶段**：✅ 完全适用，残局红桃配是救场关键

## 一、配炸弹 (`Bomb`)

**核心原理**：首配4头和同花 (`StraightFlush`)，可配6头，忌配5头。实战中逢人配成炸的比例为85%。

**平台变量名**：
- `Bomb`：炸弹
- `StraightFlush`：同花顺

### 1. 配炸多配四头炸，宜配中小不配大（51%）

**核心原理**：特别是和小3头成炸弹 (`Bomb`)，称为"麻雀变凤凰"。一般情况下在组火时配4头炸最为经济实用。

**策略**：
- 炸牌无大小，无论是大炸还是小炸，炸的作用都在于阻断对手出牌
- 在构建炸牌时更推荐选择中小点数的炸牌，而非追求高点数的大炸
- 配中小炸，留大的三带二 (`ThreeWithTwo`)，有时可以管一手牌

**平台变量名**：`ThreeWithTwo`（三带二）

### 2. 不要拆炸配同花，破二炸弹不能搭（28%）

**核心原理**：同花炸 (`StraightFlush`) 威力大，一般情况下不要强行拆炸组同花。

**原因**：拆炸组同花增强牌力，却可能带来不少副作用：
1. 单牌 (`Single`) 变多
2. 牌形变散
3. 有可能使炸弹 (`Bomb`) 的数量变少

**原则**：这与组牌的原则是相悖的。主攻不拆炸弹配同花炸，拆炸才能组同花一般是得不偿失的。

**例外**：
- 除非送对家或者管压对手最后一手牌，千万不要破俩炸弹组同花顺
- 比如有9999，需要用其中一张9和逢人配组同花，这样炸弹数量不变只是质量有所提升，不如另配头炸牌力强

**平台变量名**：
- `Single`：单张
- `Bomb`：炸弹

### 3. 配6头炸（5%）

**核心原理**：炸弹 (`Bomb`) 质量较高，闯关时十拿九稳。

### 4. 一般不配5头炸（1%）

**核心原理**：炸弹 (`Bomb`) 数量没增加，质量也不够。数量和质量两头都不占，除非情况特殊，一般不配5头炸。

### 5. 预留三张配百搭，百搭配后放后压

**核心原理**：红心配炸弹 (`Bomb`) 留到最后再使用，以便随机应变。初期保留百搭牌为后期提供更多战略变化，容易调整牌型。

### 6. 助攻可拆炸弹配同花

**策略**：助攻可以拆炸弹 (`Bomb`) 配同花 (`StraightFlush`)，为主攻提供支持。

### 7. 配同花顺大小兼顾牌型

**策略**：
- **主攻以减少手数为主**：一般配小同花 (`StraightFlush`)
- **助攻以拦截为主**：可配大同花

### 8. 不拆顺子配同花

**核心原理**：配同花 (`StraightFlush`) 导致赘牌过多，也不一定划算。

**平台变量名**：`Straight`（顺子）

## 二、补缺

**核心原理**：实战中逢人配用来补缺的比例为12%。

### 1. 配顺子 (`Straight`)

**平台变量名**：`Straight`（顺子）

**三种情况**：

1. **单张过多**：
   - 如有单张 (`Single`) 3、5、6、7但缺4，用逢人配补缺成顺，手数大大降低
   - 不然这4张赘牌不用3个炸弹 (`Bomb`) 很难护过来

2. **送对家**：
   - 明知对家手中有中大顺 (`Straight`)，或者只剩5张顺牌，组小顺子送

3. **管压对手**：
   - 对手出一顺 (`Straight`)，自己是第四家，组大顺子管压对手，最好封顶

**平台变量名**：
- `Single`：单张
- `Bomb`：炸弹

### 2. 配木板 (`ThreePair`)

**平台变量名**：`ThreePair`（三连对，木板）

**两种情况**：

1. **主攻减少手数**：
   - 33、4、55补成木板 (`ThreePair`)，减少2手赘牌
   - 木板如果登基则更理想
   - 对于主攻来讲，补缺比成炸更有利

2. **助攻尽量组成木板不要出，保留多种牌型**：
   - 一方面，可保留配炸 (`Bomb`) 的可能，必要时助力对家
   - 另一方面可以根据主攻牌型，送对子 (`Pair`) 或者三带二 (`ThreeWithTwo`)

**平台变量名**：
- `Bomb`：炸弹
- `Pair`：对子
- `ThreeWithTwo`：三带二

### 3. 配钢板 (`TwoTrips`)

**平台变量名**：`TwoTrips`（钢板）

**核心原理**：主要是主攻减少手数，或者避免对手压制，或者逼对手炸弹 (`Bomb`)。

## 三、残局运用

**核心原理**：在牌局即将失控，红心级牌是一张强有力的救场牌。在没有合适牌型对抗对手的压制牌，创造出胜利。

### 1. 配小对子 (`Pair`)

**平台变量名**：`Pair`（对子）

**两种情况**：

1. **防对手**：对手剩一张单牌 (`Single`) 或者5张牌，配小对子 (`Pair`)

2. **送对家**：对家明显要对子 (`Pair`)，配对子

**平台变量名**：`Single`（单张）

### 2. 配顺子 (`Straight`)

**平台变量名**：`Straight`（顺子）

**两种情况**：

1. **送对家**：
   - 一般助攻配顺子 (`Straight`) 送主攻
   - 主攻牌力超强也可以组顺送助攻

2. **管压对手顺子**：管压对手顺子 (`Straight`)

### 3. 配三带二 (`ThreeWithTwo`)

**平台变量名**：`ThreeWithTwo`（三带二）

**四种情况**：

1. **减少手数，便于争上游**：
   - 小对子 (`Pair`) 多，一下出两手

2. **手中就剩5张牌，恰好三带二直接走牌**

3. **管压对手三带二** (`ThreeWithTwo`)

4. **送对家**：
   - 对家剩三带二配型，手中没有合适的

**平台变量名**：`Pair`（对子）

## 注意事项

1. **配炸弹是主要用途**：实战中逢人配成炸的比例为85%
2. **补缺是次要用途**：实战中逢人配用来补缺的比例为12%
3. **残局运用是关键**：在牌局即将失控时，红心级牌是救场牌
4. **不要拆炸配同花**：除非特殊情况，不要破俩炸弹组同花顺
5. **预留三张配百搭**：红心配炸弹留到最后再使用，以便随机应变
