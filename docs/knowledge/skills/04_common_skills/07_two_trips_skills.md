# 钢板技巧

## 概述

在掼蛋游戏中，钢板（三连三）是一种特殊且重要的牌型。三同连张相当于兵器库中的宣化斧，往往一打出来就势大力沉，让人猝不及防、难以招架。

**平台变量名**：`TwoTrips`（钢板，三连三）

## 适用阶段

- **开局阶段**：✅ 适用，可以首发
- **中局阶段**：✅ 主要适用阶段
- **残局阶段**：✅ 完全适用，残局钢板是冲刺关键

## 牌型概率统计

**一盘掼蛋平均数据**：
- 平均38.3手牌，每个牌手平均会出9.8手牌
- 平均每局：
  - 单牌 (`Single`)：16手
  - 对牌 (`Pair`)：7.6手
  - 三张（包含三张牌型和三带二）(`Trips`, `ThreeWithTwo`)：7手
  - 顺子 (`Straight`)：3.3手
  - 三连对 (`ThreePair`)：1.5手
  - 钢板 (`TwoTrips`)：0.6手
  - 炸弹 (`Bomb`)：8.5手（包含四炸、多头炸和同花顺以及天王炸）

**牌型概率**：
- 单牌：49.55%
- 对子：24.77%
- 三张：8.22%
- 顺子：4.11%
- 连对：2.05%
- 钢板：1.02%
- 炸弹：5.13%
- 同花顺：2.05%

## 一、钢板的定义特点

**平台变量名**：`TwoTrips`（钢板，三连三）

**定义**：钢板是由相连3同张牌组成，可以单独作为一个组合打出6张牌。

**特点**：

1. **钢板具有灵活的拆分方式**：
   - 可以根据牌局需要调整出牌策略
   - 可以拆分成一个三带二 (`ThreeWithTwo`) 加一张单牌 (`Single`)，或者两个三张 (`Trips`) 等牌型

2. **手中有红心级牌，钢板中的任意一个三张还可以抽出来组成一个炸弹 (`Bomb`)**：
   - 红心级牌 (`H` + `main_rank`) 可以配成炸弹

3. **钢板可以一次性打出六张牌**：
   - 迅速减少手牌数量，尤其在冲刺阶段显得尤为重要

4. **特殊牌型，对家不宜管压，但是对家也不易传牌**：
   - 主动发牌基本相当于对手需要费一个炸来接
   - 但更要避免憋在手上，最后没机会上手出就可惜

**平台变量名**：
- `ThreeWithTwo`：三带二
- `Single`：单张
- `Trips`：三张（三张）
- `Bomb`：炸弹
- `H` + `main_rank`：红心级牌

## 二、钢板技巧

### 1. 牌力中等，小钢板先出

**核心原理**：没有上手机会的牌型，小钢板 (`TwoTrips`) 早点出，打出后可能会继续取得出牌权，即便对手接，也可能破坏了牌型。

### 2. 牌力很差，小钢板不出

**核心原理**：
1. 留着给对家送桥
2. 钢板出就是6张牌，避免对手走的快

### 3. 牌力强，小钢板后出

**核心原理**：但要避免憋在手上，最后没机会上手出。

**优势**：
1. **对手不好防范阻击**：产生的威胁越大，至少可以逼对手先开炸 (`Bomb`) 从而消耗其牌力
2. **出的越晚，对手可能已经拆牌**：越容易接不上
3. **可以偷袭**：有可能用来成为偷袭成为上游，手工牌我方炸后剩六张（钢板），对方看牌背面，左四右二张，对方犹豫没炸牌，偷机跑溜

**平台变量名**：`Bomb`（炸弹）

### 4. 拆分小钢板，传牌给对家

**核心原理**：当推断出对家手上有比较大的三带二 (`ThreeWithTwo`)，可以将小钢板 (`TwoTrips`) 拆成一个三带二和一个单张 (`Single`)，顺利传牌给对家，增强对家的出牌能力。

**平台变量名**：
- `ThreeWithTwo`：三带二
- `Single`：单张

### 5. 引炸策略

**核心原理**：在牌局的末段，获得牌权时一次打出6张牌，迅速减少手牌数量，可以让对手误以为你即将冲刺头游，从而慌乱出炸 (`Bomb`)。

**应用**：通过引炸，可以消耗对手的炸弹，增加我方的胜率。

**案例**：比如贡牌方，上游没希望，最后11张，忽出钢板张，急报剩5张，剩5张杂牌，引诱对家炸弹。

**平台变量名**：`Bomb`（炸弹）

### 6. 本家有小钢板，另有两个3同张，4个以上对子

**策略**：应放弃出钢板 (`TwoTrips`)，直接组出四个三带二 (`ThreeWithTwo`)，既带走小对 (`Pair`)，又减少手数。

**平台变量名**：
- `ThreeWithTwo`：三带二
- `Pair`：对子

### 7. 红配变钢板

**核心原理**：最后剩六张（AAA22+红配），判对方有一五头炸 (`Bomb`)，可一手跑光。

**平台变量名**：`Bomb`（炸弹）

## 三、应对技巧

### 1. 直接管压

**核心原理**：对方首引小钢板 (`TwoTrips`)，有现牌大钢板，可直接压，不宜开始即用红配配压（少一炸 `Bomb`）。

**平台变量名**：
- `TwoTrips`：钢板
- `Bomb`：炸弹

### 2. 下家不炸留给对家

**核心原理**：
1. 有可能对家有此类牌型
2. 最后一家是闸门，负责管控

### 3. 不接对家钢板

**核心原理**：如对家首引小钢板 (`TwoTrips`)，下家出管压，一般也不要顺接，谁打谁收。

**策略**：
- **牌力差**：留着送对家
- **牌力强**：照顾对家
- **除非差这手，顺过能头游**：可接对家钢板

### 4. 给对家传牌

**核心原理**：对家曾被对方封或炸 (`Bomb`)，对家最后剩六张，再有钢板 (`TwoTrips`) 概率极少，有可能是木板 (`ThreePair`) 或者顺子 (`Straight`)。

**案例**：根据己牌情况，己有34445556667，对家要顺，可送顺34567，留445566。

**平台变量名**：
- `Bomb`：炸弹
- `TwoTrips`：钢板
- `ThreePair`：三连对（木板）
- `Straight`：顺子

### 5. 对手出小钢板被封，自己小钢板暂时不出

**核心原理**：防止对手还有钢板 (`TwoTrips`)。

## 注意事项

1. **钢板是特殊牌型**：对家不宜管压，但是对家也不易传牌
2. **避免憋在手上**：最后没机会上手出就可惜
3. **灵活拆分**：可以根据牌局需要调整出牌策略
4. **引炸策略**：可以通过钢板引炸，消耗对手的炸弹
5. **红配变钢板**：残局阶段可以利用红心级牌配成钢板
