# 三连对技巧

## 概述

三连对（木板）相当于兵器库中的流星锤，打出去以后如同流星飞舞，令人手忙脚乱。木板属于特殊牌型，会给整个牌局带来相当大的影响，有时也可能会变成"出奇制胜"。

**平台变量名**：`ThreePair`（三连对，木板）

## 适用阶段

- **开局阶段**：✅ 适用，可以首发
- **中局阶段**：✅ 主要适用阶段
- **残局阶段**：✅ 完全适用，残局木板是偷袭利器

## 牌型概率统计

**一盘掼蛋平均数据**：
- 平均38.3手牌，每个牌手平均会出9.8手牌
- 平均每局：
  - 单牌 (`Single`)：16手
  - 对牌 (`Pair`)：7.6手
  - 三张（包含三张牌型和三带二）(`Trips`, `ThreeWithTwo`)：7手
  - 顺子 (`Straight`)：3.3手
  - 三连对 (`ThreePair`)：1.5手
  - 钢板 (`TwoTrips`)：0.6手
  - 炸弹 (`Bomb`)：8.5手（包含四炸、多头炸和同花顺以及天王炸）

## 一、木板的定义特点

**平台变量名**：`ThreePair`（三连对，木板）

**特点**：

1. **出牌速度快**：
   - 木板特点一次可以出6张牌，有效减少牌的手数，达到快跑目的

2. **把小的三个对子组织到一起，变废为宝**：
   - 将三个小对子 (`Pair`) 组合成三连对，提高牌型价值

3. **特殊牌型，不易管压**：
   - 打出后取得继续领出牌权的可能性会比较大
   - 对手强行变牌阻击，可能会破坏其原有的牌型，打乱进攻计划
   - 在残局阶段打出的三连对，经常是偷袭而成为上游的利器

4. **可拆可变，灵活性能高**：
   - 可以根据牌局需要拆分成对子或其他牌型

5. **对称原理，你有我有咱都有**：
   - 两家打出木板 (`ThreePair`)，谨防第三家或四家都有木板

**平台变量名**：`Pair`（对子）

## 二、木板技巧

### 1. 首引一般不轻易出木板

**核心原理**：进贡方，牌力中，但不一定下游，首引一般不轻易出木板 (`ThreePair`)。

**策略**：
- 首引木板，对手会变化牌型来进行阻击
- 后面打出来，对手用变牌来进行防范可能性变小，产生威胁变大，达到先逼对手开炸 (`Bomb`) 目的
- 若有中、大木板，更不应首打，留着能压对方上手

**平台变量名**：`Bomb`（炸弹）

### 2. 不接对家木板

**核心原理**：
1. **对家首引木板 (`ThreePair`)，手中即便有木板，一般也不要顺接**：
   - 己若牌力差，留着送对家
   - 除非差这手，顺过能头游，可接对家木板

2. **下家管压，手中即便有大木板，定位助攻，一般也不要顺接**：
   - 谁打谁收，后期可送

### 2.1 先保留一个三连对

**核心原理**：组牌时，先保留一个三连对 (`ThreePair`)，方便配合对家或根据牌局形势灵活调整。

**策略要点**：
- 不要急于拆开三连对
- 保留三连对，方便配合对家
- 根据牌局形势，灵活决定是否拆开或使用三连对

**注意事项**：
- 如果对家需要三连对，可以送三连对
- 如果对家跟了一手三连对，说明对家可能需要三连对或相关牌型

### 3. 还贡拆三张组木板

**核心原理**：4455566，防止对手方成炸弹 (`Bomb`)，拆555还贡5。

**平台变量名**：`Bomb`（炸弹）

### 4. 改变牌型组木板阻对手

**策略**：

1. **拆三带二 (`ThreeWithTwo`) 组成木板 (`ThreePair`) 阻对手**：
   - 对方首引223344，己可组99101010JJ，多单10

2. **变顺为木板**：
   - 对方首出顺 (`Straight`)，被对家炸 (`Bomb`)，己原有两把小顺，可变一至两个木板 (`ThreePair`)

3. **变木板为顺**：
   - 对方首出木板 (`ThreePair`)，被对家炸 (`Bomb`)，己有两个小木板，可变两把小顺 (`Straight`)
   - 也可配合对家牌型变换

**平台变量名**：
- `ThreeWithTwo`：三带二
- `ThreePair`：三连对（木板）
- `Straight`：顺子
- `Bomb`：炸弹

### 5. 木板直接封到顶

**核心原理**：对方首引小木板 (`ThreePair`)，本家有1010JJQQKK，要封就封大JJQQKK，防对方回手JJQQKK压。

### 6. 送对家木板

**策略**：

1. **拆三带二 (`ThreeWithTwo`) 组成木板 (`ThreePair`) 送对家**：
   - 对家首引223344，被对方封或炸 (`Bomb`)，伺机送445566，自己不能上游，不怕多单5

2. **用红心配组木板送对家**：
   - 对家首引木板 (`ThreePair`) 曾被对方封或炸 (`Bomb`)，对家最后剩六张，可能还是木板
   - 本家可用百搭配5，组445566送对家

3. **拆炸弹 (`Bomb`) 组木板 (`ThreePair`)**：
   - 如334455556677，拆一个炸弹，组两套木板

**平台变量名**：
- `ThreeWithTwo`：三带二
- `ThreePair`：三连对（木板）
- `Bomb`：炸弹

### 7. 诱骗炸弹

**核心原理**：贡牌方，上游没希望，最后11张，忽出木板 (`ThreePair`)，急报剩5张杂牌，骗对手炸弹 (`Bomb`)，方便对家走牌。

**平台变量名**：`Bomb`（炸弹）

### 8. 利用特殊牌型摆尾

**核心原理**：尾牌剩七张木板 (`ThreePair`) 加一张，如7788999，恐对方有大三带二 (`ThreeWithTwo`)，判对方已无炸 (`Bomb`)，可出木板，留单9上游。

**平台变量名**：
- `ThreePair`：三连对（木板）
- `ThreeWithTwo`：三带二
- `Bomb`：炸弹

### 9. 故意牌摆偷跑

**核心原理**：手工牌我方出牌后剩6张，左5右1张，或者上4下2，对方看牌背面不是一手牌，犹豫没有炸 (`Bomb`)，偷机跑溜。

**平台变量名**：`Bomb`（炸弹）

## 注意事项

1. **首引一般不轻易出木板**：后面打出来，威胁更大

5. **残局阶段是偷袭利器**：在残局阶段打出的三连对，经常是偷袭而成为上游的利器
