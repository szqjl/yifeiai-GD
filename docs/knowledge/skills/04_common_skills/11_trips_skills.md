# 三张技巧

## 概述

三张（三张）相当于兵器库中的三股托天叉，一出手就不同凡响，让人摸不着头脑，连续打出几手，杀伤力就比打对子 (`Pair`) 还要大。打三张的特点就是怪，俗话说"三张、毒怪怪"。连续打出多手三张牌，而两名对手还一张牌未出，又没有及时开炸 (`Bomb`) 阻拦，就形成了接近冲刺的局面。

**定位说明**：三张主要用于助攻打法，但主攻也可以使用三张。掼蛋就是这么神奇，在不同的情况下，助攻可转换成主攻。

**平台变量名**：
- `Trips`：三张（三张）
- `Pair`：对子
- `Bomb`：炸弹

## 适用阶段

- **开局阶段**：✅ 适用，可以首发
- **中局阶段**：✅ 主要适用阶段
- **残局阶段**：✅ 完全适用，残局三张是偷袭利器

## 一、三张的定义特点

**平台变量名**：`Trips`（三张，三张）

**特点**：

1. **三张的主要作用，破坏对手牌型的助攻打法**：
   - 主要用于助攻，在牌力不强，不大可能争到上游时采用
   - 牌手接连打出三张 (`Trips`)，逼对手破坏三带二 (`ThreeWithTwo`) 等优势牌型来进行阻击
   - 打乱对手原有的组牌与进攻计划，为对家在牌局中争得有利态势
   - **注意**：主攻也可以使用三张，掼蛋就是这么神奇，在不同的情况下，助攻可转换成主攻

2. **过早暴露牌路特点**：
   - 出三张 (`Trips`) 表明牌型中基本没有顺子 (`Straight`)
   - 对子 (`Pair`) 也比较少，炸 (`Bomb`) 不会很多
   - 单张 (`Single`) 有可能会比较多，其牌力并不会很强

3. **灵活性强**：
   - 三张既可以带对子 (`Pair`) 打出整手牌型 (`ThreeWithTwo`)
   - 也可以与红桃级牌配成炸弹 (`Bomb`)
   - 还可以在参加组成顺子 (`Straight`) 时余下一对牌

4. **可拆分对子单张**：
   - 可以根据牌局需要拆分成对子或单张

**平台变量名**：
- `ThreeWithTwo`：三带二
- `Straight`：顺子
- `Pair`：对子
- `Bomb`：炸弹
- `Single`：单张

## 二、出牌技巧

### 1. 手中就是"三张"牌型

**核心原理**：基本牌型只能打"三张" (`Trips`)，打出来多数有回手、甚至对手大牌控制开炸 (`Bomb`) 出牌。

### 2. "三张"小，对子大

**核心原理**：让对手误以为没有对子 (`Pair`)，引起对手用大的三个头压牌，打对后遭遇出牌人大对上手。从控牌的角度说，"三张" (`Trips`) 上手控一牌，若出对控一手，不吃亏，比出"三带二" (`ThreeWithTwo`) 的牌型让对手完全控一手要好。

**平台变量名**：
- `Pair`：对子
- `Trips`：三张（三张）
- `ThreeWithTwo`：三带二

### 3. 搅局

**核心原理**：
- 牌不好，应该打"三带二" (`ThreeWithTwo`) 的牌，为不给对手带牌，打"三张" (`Trips`)，让对手多出对子 (`Pair`)
- 牌力弱的"三张"，打出来后就不要，打对也不要

### 4. 拆三张

**策略**：
1. **对子是弱项，可用三A（或三K）+红配**：封对方两次大对 (`Pair`)
2. **我方牌弱，不能上游，对家牌少，明确要对**：己已无小对，可用红配+破333，送两次小对 (`Pair`)，救对家
3. **拆三张 (`Trips`) 组顺子 (`Straight`)**：送对家

**平台变量名**：
- `Pair`：对子
- `Straight`：顺子

### 5. 溜牌

**核心原理**：下家剩两张，判断是大对 (`Pair`)，己剩1单 (`Single`) + 1对 (`Pair`) + 红配，可组三个头 (`Trips`) 跑溜。

**平台变量名**：
- `Pair`：对子
- `Single`：单张
- `Trips`：三张

### 6. 组牌

**策略**：
1. **三个+配+单（成夯 `ThreeWithTwo`）**：
   - 如己剩十张，先出一小夯，后出一大夯（大三+配+单）
2. **剩五张牌（三+配+单），有出夯顺跑**：
   - 为控下家单 (`Single`)，可出三+配+单脱手或救对家小夯 (`ThreeWithTwo`)

**平台变量名**：
- `ThreeWithTwo`：三带二（夯）
- `Single`：单张

## 三、应对技巧

### 1. 对家前打过三张，最后剩三张

**核心原理**：即判一手牌。己有小三 (`Trips`)，直接送；没有三个头，又无力做上游，可用红配配小对 (`Pair`) 成小三。

**平台变量名**：
- `Trips`：三张
- `Pair`：对子

### 2. 出牌顺序中大小

**核心原理**：对家剩三张，不能从最小三个头出，防下家封或炸 (`Bomb`)，等对手拦截再出大的三张 (`Trips`)，最后出小的三张。

**平台变量名**：`Bomb`（炸弹）

### 3. 先出三张，后有三带二

**核心原理**：对家前打三张 (`Trips`)，不是一对没有（对大或对少），如对家最后剩九张，己应直接去送三带二 (`ThreeWithTwo`)，送三张，对家反而多一对 (`Pair`)。

**平台变量名**：
- `ThreeWithTwo`：三带二
- `Pair`：对子

### 4. 红配组三张

**核心原理**：我方小对多，对方出333444，55+666+红配，可压对方，减小对 (`Pair`)。

### 5. 顶大牌拦截

**核心原理**：对手剩七张，恐有一炸 (`Bomb`)，外加三个头 (`Trips`)，需顶大三个头，直接顶死。

**平台变量名**：
- `Bomb`：炸弹
- `Trips`：三张

### 6. 开炸

**核心原理**：上家曾多次打过三张 (`Trips`)，最后剩4张，按理枪不打四，本家多两小单 (`Single`)，此时需判三炸四，否则对方出三留单。

**平台变量名**：
- `Single`：单张

### 7. 骗炸

**核心原理**：最后剩六张（333+KK+4），先出333，尾留虚三张，对手认为一手牌，没准会开炸 (`Bomb`)。

**平台变量名**：`Bomb`（炸弹）

## 注意事项

1. **三张主要用于助攻打法**：在牌力不强时采用，破坏对手牌型。但主攻也可以使用三张，掼蛋就是这么神奇，在不同的情况下，助攻可转换成主攻
2. **过早暴露牌路特点**：出三张表明牌型中基本没有顺子，对子也比较少
3. **灵活性强**：三张既可以带对子打出整手牌型，也可以与红桃级牌配成炸弹
4. **"三张、毒怪怪"**：连续打出多手三张牌，杀伤力很大
5. **先出三张，后有三带二**：对家前打三张，后面要注意送三带二
