# 组牌技巧

## 概述

组牌技巧是掼蛋开局阶段的核心技能，指拿到一副牌后，如何组合手中的牌，让它发挥最大效力，或者给对方提供支持。组牌的好坏直接影响整局游戏的胜负。

## 适用阶段

- **开局阶段**：? 主要适用于开局阶段的起手组牌
- **中局阶段**：?? 部分适用，但开局组牌是基础
- **残局阶段**：? 不适用

## 核心原则

1. **减少轮次为中心**：组牌要以减少轮次为中心
2. **灵活组合**：牌型不组死，重在看走势，静态组合加上动态变化才是取胜之道
3. **经济实用**：配火要经济实用，避免因配错一手牌而失去整局胜利

## 组牌技巧详解

### 1. 开局先想通，能否当主攻

**技巧说明**：
起完牌后，首先根据牌力之强弱定位。牌力很强就做主攻，争取上游；牌力太弱就做助攻。

**关键要点**：
- 牌力评估是组牌的前提
- 主攻和助攻的组牌策略不同
- 记住，你不是一个人在战斗，要考虑团队配合

**应用场景**：
- 起手牌力评估
- 角色定位（主攻/助攻）

### 2. 组牌有诀窍，确保轮次少

**技巧说明**：
组牌要以减少轮次为中心。

**关键要点**：
- 轮次越少，出牌效率越高
- 优先组合能减少手数的牌型
- 避免制造过多单张

**应用场景**：
- 所有组牌决策的基础原则

### 3. 配火多配四头火，宜配中小不配大

**技巧说明**：
配四头火（4张炸弹）最为经济实用。如果违背配火的经济原则，往往会因配错一手牌而失去整局的胜利。

**关键要点**：
- 四头火（4张炸弹）最经济实用
- 宜配中小牌点，不配大牌点
- 经济原则是配火的核心

**应用场景**：
- 有红桃配（逢人配）时配炸弹
- 选择配什么牌点的炸弹

**平台术语**：
- 四头火 = 4张炸弹 (`Bomb`)
- 红桃配 = 红心级牌（逢人配，`H` + `curRank`）

### 4. 破二炸弹不能搭

**技巧说明**：
通常不必为了配一个同花顺，而拆掉两个四头炸弹。不过，紧急关头除外。

**关键要点**：
- 两个四头炸弹的价值大于一个同花顺
- 不要轻易拆炸弹配同花顺
- 紧急关头可以例外

**应用场景**：
- 有多个四头炸弹时
- 考虑是否拆炸弹配同花顺

**平台术语**：
- 四头炸弹 = 4张炸弹 (`Bomb`)
- 同花顺 = `StraightFlush`

### 5. 预留3个配百搭，百搭配3放后压

**技巧说明**：
百搭指的是红心级牌（逢人配）。用红心级牌配的炸弹，留到后面使用，以便随机应变。

**关键要点**：
- 百搭 = 红心级牌（逢人配，`H` + `curRank`）
- 用红桃配配的炸弹，留到后面使用
- 便于随机应变

**应用场景**：
- 有多个红桃配时
- 决定哪些炸弹先出，哪些后出

**平台术语**：
- 百搭 = 红心级牌（逢人配，`H` + `curRank`）
- 配3 = 配成3张炸弹（实际是4张，包括红桃配）

### 6. 配了单张算百搭，百搭单走算白搭

**技巧说明**：
如果用红桃配配了单张，就浪费了百搭的功能。百搭单独走（当作单张出）也是浪费。

**关键要点**：
- 红桃配不要配单张
- 红桃配不要单独当单张出
- 要充分利用红桃配的搭配功能

**应用场景**：
- 避免浪费红桃配
- 合理使用逢人配

### 7. 同花优先炸弹后，再配顺子三带二

**技巧说明**：
若有红心级牌（逢人配），须首配同花顺、次配炸弹、再次补缺减手数。

**关键要点**：
- 配牌优先级：同花顺 > 炸弹 > 顺子/三带二
- 首配同花顺（`StraightFlush`）
- 次配炸弹（`Bomb`）
- 再次配顺子（`Straight`）或三带二（`ThreeWithTwo`）来补缺减手数

**应用场景**：
- 有红桃配时的配牌顺序
- 决定优先配什么牌型

**平台术语**：
- 同花顺 = `StraightFlush`
- 炸弹 = `Bomb`
- 顺子 = `Straight`
- 三带二 = `ThreeWithTwo`

### 8. 顺子多了有点累，单张多了活受罪

**技巧说明**：
顺子多了会增加手数，单张多了更是不利。

**关键要点**：
- 不要组太多顺子
- 避免制造过多单张
- 平衡各种牌型的组合

**应用场景**：
- 避免过度组顺子
- 控制单张数量

### 9. 手上断张和单张，外面九成为炸弹

**技巧说明**：
如果手上有断张（缺少的牌点）和单张，外面这些牌点很可能成为炸弹。

**关键要点**：
- 断张和单张往往对应外面的炸弹
- 要警惕这些牌点
- 记牌时要特别注意

**应用场景**：
- 记牌分析
- 判断外面可能的炸弹

### 10. 化零为整，不输有门

**技巧说明**：
当手中的牌比较零散时，一定要尽量整合。

**关键要点**：
- 零散的牌要尽量整合
- 减少手数
- 提高出牌效率

**应用场景**：
- 手牌零散时的整合策略

### 11. 牌型不组死，重在看走势

**技巧说明**：
如果把牌组死，守株待兔，就忘记了还有其它牌型的灵活变化，会坐失接牌或控牌的良机。静态组合加上动态变化才是取胜之道。

**关键要点**：
- 不要组死牌型
- 要保持灵活性
- 静态组合 + 动态变化

**应用场景**：
- 避免过度固定牌型
- 保持组牌的灵活性

### 12. 组顺生两单，肯定没眼光

**技巧说明**：
如果由于组顺子而造出两张以上单牌，则应放弃。

**关键要点**：
- 组顺子不要制造两张以上单牌
- 如果组顺子会增加单张，应该放弃
- 减少单张是组牌的重要原则

**应用场景**：
- 决定是否组顺子
- 评估组顺子的代价

**平台术语**：
- 顺子 = `Straight`
- 单张 = `Single`

### 13. 要让牌技高，记牌第一招

**技巧说明**：
如果不记牌，就会在起跑线上输给记牌的对手。

**关键要点**：
- 记牌是组牌的基础
- 不记牌会处于劣势
- 记牌有助于判断和决策

**应用场景**：
- 开局阶段的记牌
- 为组牌提供信息支持

### 14. 无论小与大，坚持去单化

**技巧说明**：
无论牌点大小，都要坚持去单化（减少单张）。

**关键要点**：
- 去单化是组牌的核心原则
- 无论牌点大小都要坚持
- 减少单张数量

**应用场景**：
- 所有组牌决策
- 优先减少单张

### 15. 组顺去小单，效果不一般

**技巧说明**：
组顺子若能把小单吸收到顺子里面，就减少了手数。

**关键要点**：
- 组顺子要优先吸收小单
- 减少手数
- 提高出牌效率

**应用场景**：
- 有顺子和小单时
- 决定是否组顺子

**平台术语**：
- 顺子 = `Straight`
- 单张 = `Single`

### 16. 要想牌有劲，轻易不组顺

**技巧说明**：
如果组顺破坏其他更为有利的搭配方法，应该放弃。

**关键要点**：
- 不要轻易组顺子
- 如果组顺破坏其他有利搭配，应该放弃
- 优先选择更有利的搭配

**应用场景**：
- 评估组顺子的利弊
- 选择最优组牌方案

**平台术语**：
- 顺子 = `Straight`

### 17. 手数要减少，拆牌能奏效

**技巧说明**：
如果手牌有3个三同张和1个对子，不妨考虑从其中1个三同张中拆出一张单牌顺过，留出2个三带二牌型，以减少手数。

**关键要点**：
- 拆牌可以减少手数
- 3个三张 + 1个对子 → 拆1个三张 → 2个三带二
- 灵活拆牌是组牌技巧

**应用场景**：
- 有多个三张和对子时
- 通过拆牌减少手数

**平台术语**：
- 三同张 = 三张 (`Trips`)
- 对子 = `Pair`
- 三带二 = `ThreeWithTwo`

## 组牌优先级总结

### 红桃配（逢人配）的配牌优先级

1. **首配同花顺** (`StraightFlush`)
2. **次配炸弹** (`Bomb`) - 优先配四头火（4张），宜配中小不配大
3. **再次配顺子** (`Straight`) 或 **三带二** (`ThreeWithTwo`) - 补缺减手数

### 组牌原则优先级

1. **减少轮次** - 组牌的核心目标
2. **去单化** - 无论小与大，坚持去单化
3. **化零为整** - 零散牌要整合
4. **灵活组合** - 牌型不组死，重在看走势

## 常见错误

- ? **配错一手牌**：违背配火的经济原则，因配错一手牌而失去整局胜利
- ? **破二炸弹配同花顺**：为了配一个同花顺而拆掉两个四头炸弹（紧急关头除外）
- ? **百搭配单张**：用红桃配配单张，浪费百搭功能
- ? **百搭单走**：红桃配单独当单张出，算白搭
- ? **组顺生两单**：由于组顺子而造出两张以上单牌
- ? **组死牌型**：把牌组死，失去灵活性
- ? **轻易组顺**：组顺破坏其他更为有利的搭配方法
